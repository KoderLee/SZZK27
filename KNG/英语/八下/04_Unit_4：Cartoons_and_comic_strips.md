## 知识点2：重点句型

### 内容
**结果状语从句**：
- so...that... 如此……以至于……
- such...that... 如此……以至于……

**重点句型**：
- The cartoon is so interesting that I have watched it many times.
  这部卡通片太有趣了，我已经看了很多遍。
- It is such a funny cartoon that everyone likes it.
  这是一部如此有趣的卡通片，以至于每个人都喜欢它。
- The character is so popular that he became famous all over the world.
  这个角色太受欢迎了，以至于他在全世界都很有名。

## 知识点3：语法——结果状语从句

### 内容
**so...that... 的用法**：
1. so + 形容词/副词 + that
   - He ran so fast that I couldn't catch up with him.
2. so + 形容词 + a/an + 可数名词单数 + that
   - It was so interesting a book that I read it twice.
3. so + many/much/few/little + 名词 + that
   - There were so many people that we couldn't get in.

**such...that... 的用法**：
1. such + a/an + 形容词 + 可数名词单数 + that
   - It is such a beautiful day that we should go out.
2. such + 形容词 + 不可数名词 + that
   - It was such fine weather that we went camping.
3. such + 形容词 + 可数名词复数 + that
   - They are such good students that we all like them.

**so that 引导的目的状语从句**：
- I got up early so that I could catch the early bus.
