## 知识点1：常见的酸和碱

### 内容
**常见的酸**：
- 盐酸（HCl）：无色液体，有刺激性气味，易挥发
- 硫酸（H₂SO₄）：无色黏稠液体，不易挥发，有吸水性、脱水性
- 硝酸（HNO₃）：无色液体，有刺激性气味，易挥发，有强氧化性

**常见的碱**：
- 氢氧化钠（NaOH）：白色固体，易潮解，易溶于水，有强腐蚀性（烧碱、火碱、苛性钠）
- 氢氧化钙（Ca(OH)₂）：白色粉末，微溶于水（熟石灰、消石灰）
- 氨水（NH₃·H₂O）：无色液体，有刺激性气味

**浓硫酸的特性**：
- 吸水性：可作干燥剂（干燥中性或酸性气体）
- 脱水性：能将有机物中的氢氧元素按水的组成比脱去
- 强氧化性
- 稀释时放热

**浓硫酸的稀释**：
- 酸入水，沿器壁，慢慢倒，不断搅
- 切不可将水倒入浓硫酸中

## 知识点2：酸的化学性质

### 内容
**酸与指示剂反应**：
- 使紫色石蕊溶液变红
- 不能使无色酚酞溶液变色

**酸与活泼金属反应**（置换反应）：
- Mg + 2HCl = MgCl₂ + H₂↑
- Zn + H₂SO₄ = ZnSO₄ + H₂↑
- Fe + 2HCl = FeCl₂ + H₂↑（生成亚铁盐，溶液呈浅绿色）

**酸与金属氧化物反应**（复分解反应）：
- Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O（除铁锈，溶液变黄）
- CuO + H₂SO₄ = CuSO₄ + H₂O（黑色固体溶解，溶液变蓝）

**酸与碱反应**（中和反应，复分解反应）：
- HCl + NaOH = NaCl + H₂O
- H₂SO₄ + Ca(OH)₂ = CaSO₄ + 2H₂O

**酸与某些盐反应**（复分解反应）：
- 2HCl + Na₂CO₃ = 2NaCl + H₂O + CO₂↑
- HCl + AgNO₃ = AgCl↓ + HNO₃
- H₂SO₄ + BaCl₂ = BaSO₄↓ + 2HCl

- 复分解反应的条件：生成沉淀、气体或水

## 知识点3：碱的化学性质

### 内容
**碱与指示剂反应**：
- 使紫色石蕊溶液变蓝
- 使无色酚酞溶液变红

**碱与非金属氧化物反应**：
- 2NaOH + CO₂ = Na₂CO₃ + H₂O（吸收CO₂，变质原理）
- Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O（检验CO₂）
- 2NaOH + SO₂ = Na₂SO₃ + H₂O（吸收SO₂）

**碱与酸反应**（中和反应）：
- 见酸的化学性质

**碱与某些盐反应**（复分解反应）：
- 2NaOH + CuSO₄ = Cu(OH)₂↓ + Na₂SO₄（蓝色沉淀）
- 3NaOH + FeCl₃ = Fe(OH)₃↓ + 3NaCl（红褐色沉淀）
- Ca(OH)₂ + Na₂CO₃ = CaCO₃↓ + 2NaOH（工业制烧碱）

## 知识点4：中和反应

### 内容
**中和反应定义**：
- 酸与碱作用生成盐和水的反应
- 实质：H⁺ + OH⁻ = H₂O

**中和反应的应用**：
- 调节土壤酸碱性
- 处理工厂废水
- 医疗上治疗胃酸过多（Al(OH)₃、Mg(OH)₂）
- 蚊虫叮咬后涂碱性物质止痒

**中和反应的实验探究**：
- 指示剂：酚酞或石蕊
- 操作：向碱溶液中滴加酸，边滴边搅拌
- 终点判断：溶液颜色恰好褪去（酚酞）或恰好变色（石蕊）

**pH与溶液酸碱性**：
- pH < 7：酸性，pH越小酸性越强
- pH = 7：中性
- pH > 7：碱性，pH越大碱性越强

**pH的测定**：
- pH试纸：用玻璃棒蘸取待测液，滴在pH试纸上，与标准比色卡对比
- 注意事项：不能将pH试纸直接浸入待测液；不能先用水润湿
