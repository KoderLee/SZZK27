## 知识点1：生活中常见的盐

### 内容
**氯化钠（NaCl）**：
- 俗称：食盐
- 用途：调味品、制生理盐水、化工原料、融雪剂
- 来源：海水晒盐、盐湖、盐井

**碳酸钠（Na₂CO₃）**：
- 俗称：纯碱、苏打
- 性质：白色固体，易溶于水，水溶液呈碱性
- 用途：制玻璃、造纸、纺织、洗涤剂

**碳酸氢钠（NaHCO₃）**：
- 俗称：小苏打
- 性质：白色固体，可溶于水，水溶液呈弱碱性
- 用途：焙制糕点（发酵粉主要成分）、治疗胃酸过多、灭火剂

**碳酸钙（CaCO₃）**：
- 存在：大理石、石灰石、蛋壳、贝壳
- 性质：白色固体，难溶于水
- 用途：建筑材料、补钙剂、工业制CO₂

**碳酸盐的检验**：
- 加入稀盐酸，产生能使澄清石灰水变浑浊的气体
- CO₃²⁻ + 2H⁺ = H₂O + CO₂↑

## 知识点2：盐的化学性质

### 内容
**盐与金属反应**（置换反应）：
- Fe + CuSO₄ = FeSO₄ + Cu
- Cu + 2AgNO₃ = Cu(NO₃)₂ + 2Ag
- 条件：金属活动性顺序中，前面的金属置换后面的金属（K、Ca、Na除外）

**盐与酸反应**（复分解反应）：
- CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
- AgNO₃ + HCl = AgCl↓ + HNO₃
- BaCl₂ + H₂SO₄ = BaSO₄↓ + 2HCl

**盐与碱反应**（复分解反应）：
- CuSO₄ + 2NaOH = Cu(OH)₂↓ + Na₂SO₄
- FeCl₃ + 3NaOH = Fe(OH)₃↓ + 3NaCl
- Na₂CO₃ + Ca(OH)₂ = CaCO₃↓ + 2NaOH
- 条件：反应物都可溶，生成沉淀

**盐与盐反应**（复分解反应）：
- NaCl + AgNO₃ = AgCl↓ + NaNO₃
- Na₂SO₄ + BaCl₂ = BaSO₄↓ + 2NaCl
- Na₂CO₃ + CaCl₂ = CaCO₃↓ + 2NaCl
- 条件：反应物都可溶，生成沉淀

## 知识点3：复分解反应

### 内容
**复分解反应定义**：
- 两种化合物互相交换成分，生成另外两种化合物的反应
- 通式：AB + CD = AD + CB

**复分解反应的条件**：
- 生成沉淀
- 生成气体
- 生成水（中和反应是特例）

**常见沉淀**：
- 白色：CaCO₃、BaCO₃、AgCl、BaSO₄、Mg(OH)₂、Al(OH)₃
- 蓝色：Cu(OH)₂
- 红褐色：Fe(OH)₃

**不溶于稀硝酸的沉淀**：
- AgCl（白色）
- BaSO₄（白色）

**离子共存**：
- 能反应生成沉淀、气体或水的离子不能共存
- 常见不能共存的离子对：H⁺和OH⁻、H⁺和CO₃²⁻、Ca²⁺和CO₃²⁻、Ba²⁺和SO₄²⁻、Ag⁺和Cl⁻等

- 中和反应是复分解反应的特例
- 碱与酸性氧化物反应不属于复分解反应

## 知识点4：化学肥料

### 内容
**化肥的分类**：
- 氮肥：含氮元素（尿素CO(NH₂)₂、氨水NH₃·H₂O、铵盐、硝酸盐）
- 磷肥：含磷元素（磷矿粉、钙镁磷肥、过磷酸钙）
- 钾肥：含钾元素（K₂SO₄、KCl）
- 复合肥：含N、P、K中两种或两种以上（KNO₃、NH₄H₂PO₄）

**化肥的作用**：
- 氮肥：促进叶色浓绿，枝叶茂盛
- 磷肥：促进根系发达，籽粒饱满
- 钾肥：促进茎秆粗壮，增强抗病虫害能力

**化肥的鉴别**：
- 看外观：氮肥、钾肥为白色晶体，磷肥为灰白色粉末
- 加水溶解：氮肥、钾肥易溶，磷肥大多不溶
- 加熟石灰研磨：铵态氮肥（铵盐）会放出有刺激性气味的氨气

**铵态氮肥的使用注意事项**：
- 不能与碱性物质（如熟石灰、草木灰）混合施用
- 否则会放出氨气，降低肥效：2NH₄Cl + Ca(OH)₂ = CaCl₂ + 2H₂O + 2NH₃↑
