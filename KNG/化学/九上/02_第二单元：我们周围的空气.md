## 知识点1：空气

### 内容
**空气的组成**（体积分数）：
| 成分 | 体积分数 | 性质和用途 |
|------|----------|------------|
| 氮气(N₂) | 约78% | 化学性质稳定，作保护气、制氮肥 |
| 氧气(O₂) | 约21% | 支持燃烧、供给呼吸 |
| 稀有气体 | 约0.94% | 化学性质极不活泼，作保护气、制电光源 |
| 二氧化碳(CO₂) | 约0.03% | 光合作用原料 |
| 其他气体和杂质 | 约0.03% | - |

**空气中氧气含量的测定**：
- **原理**：利用红磷燃烧消耗氧气，使瓶内压强减小
- **反应**：4P + 5O₂ → 2P₂O₅（点燃）
- **现象**：红磷燃烧，产生大量白烟，冷却后打开止水夹，水倒流入集气瓶约占1/5
- **结论**：氧气约占空气体积的1/5
- **注意事项**：
  - 红磷要过量（耗尽氧气）
  - 装置气密性要好
  - 冷却至室温再打开止水夹
  - 不能用硫、碳代替红磷（生成气体）

## 知识点2：氧气

### 内容
**氧气的物理性质**：
- 无色、无味、气体
- 密度比空气略大
- 不易溶于水
- 液态和固态呈淡蓝色

**氧气的化学性质**（助燃性、氧化性）：

| 反应物质 | 在空气中 | 在氧气中 | 化学方程式 |
|----------|----------|----------|------------|
| 木炭 | 红热 | 发出白光 | C + O₂ → CO₂ |
| 硫 | 淡蓝色火焰 | 蓝紫色火焰 | S + O₂ → SO₂ |
| 铁丝 | 不燃烧 | 火星四射 | 3Fe + 2O₂ → Fe₃O₄ |
| 磷 | 黄白色火焰 | 白光 | 4P + 5O₂ → 2P₂O₅ |
| 镁条 | 耀眼白光 | 更耀眼白光 | 2Mg + O₂ → 2MgO |

**注意事项**：
- 硫在氧气中燃烧：集气瓶底放水吸收SO₂
- 铁丝在氧气中燃烧：集气瓶底放少量水或细沙，防止炸裂

**氧气的用途**：
- 供给呼吸（医疗、潜水）
- 支持燃烧（炼钢、气焊）

## 知识点3：制取氧气

### 内容
**实验室制取氧气**：

| 方法 | 反应原理 | 装置类型 | 收集方法 |
|------|----------|----------|----------|
| 加热高锰酸钾 | 2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑ | 固体加热型 | 排水法或向上排空气法 |
| 加热氯酸钾 | 2KClO₃ → 2KCl + 3O₂↑（MnO₂催化） | 固体加热型 | 同上 |
| 分解过氧化氢 | 2H₂O₂ → 2H₂O + O₂↑（MnO₂催化） | 固液常温型 | 同上 |

**催化剂**：
- 定义：在化学反应里能改变其他物质的化学反应速率，而本身的质量和化学性质在反应前后都没有发生变化的物质
- 特点：一变（速率）两不变（质量、化学性质）
- 注意：催化剂只改变化学反应速率，不影响生成物质量

**实验装置**：
- 发生装置选择依据：反应物状态和反应条件
- 收集装置选择依据：气体的密度和溶解性
  - 排水法：适用于不易溶于水且不与水反应的气体（较纯净）
  - 向上排空气法：适用于密度比空气大的气体
  - 向下排空气法：适用于密度比空气小的气体

**检验和验满**：
- 检验：将带火星的木条伸入集气瓶内，木条复燃
- 验满：将带火星的木条放在集气瓶口，木条复燃

**实验步骤**（以高锰酸钾为例）：
1. 查：检查装置气密性
2. 装：装入药品
3. 定：固定装置（试管口略向下倾斜）
4. 点：点燃酒精灯加热
5. 收：收集气体
6. 离：撤离导管
7. 熄：熄灭酒精灯

**注意事项**：
- 试管口略向下倾斜（防止冷凝水倒流炸裂试管）
- 试管口塞一团棉花（防止高锰酸钾粉末进入导管）
- 先预热再集中加热
- 实验结束先撤导管后熄灯（防止水倒吸）

**工业制氧气**：
- 方法：分离液态空气法（物理变化）
- 原理：利用氮气和氧气沸点不同
