## 知识点2：二氧化碳制取的研究

### 内容
**实验室制法**：
- 原理：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- 装置：固液常温型
- 收集：向上排空气法（密度比空气大，能溶于水）
- 检验：通入澄清石灰水，变浑浊
- 验满：燃着的木条放在瓶口，熄灭

**注意事项**：
- 不用浓盐酸（挥发性强，气体不纯）
- 不用硫酸（生成微溶硫酸钙，反应停止）
- 不用碳酸钠（反应太快，不易控制）

**工业制法**：
- CaCO₃ → CaO + CO₂↑（高温煅烧石灰石）

## 知识点3：二氧化碳和一氧化碳

### 内容
**二氧化碳（CO₂）**：
- 物理性质：无色无味气体，密度比空气大，能溶于水，固态叫干冰
- 化学性质：
  - 不燃烧，不支持燃烧
  - 与水反应：CO₂ + H₂O → H₂CO₃
  - 与石灰水反应：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
- 用途：灭火、制冷剂、光合作用、化工原料

**一氧化碳（CO）**：
- 物理性质：无色无味气体，密度比空气略小，难溶于水
- 化学性质：
  - 可燃性：2CO + O₂ → 2CO₂（蓝色火焰）
  - 还原性：CO + CuO → Cu + CO₂（加热）
  - 毒性：与血红蛋白结合，使人缺氧
- 用途：燃料、冶炼金属

**CO与CO₂的鉴别**：
- 通入澄清石灰水
- 通入紫色石蕊溶液
- 通过灼热的氧化铜

**核心概念**：
- 碳单质的性质和用途 ⭐⭐⭐⭐

**核心方程式**：
- CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
- C + 2CuO → 2Cu + CO₂↑（高温）
- CO + CuO → Cu + CO₂（加热）

**高频考点**：
- 碳单质性质差异原因

**易错点**：
- 碳单质物理性质差异是碳原子排列方式不同
- 制CO₂不能用硫酸和碳酸钠
- CO有毒，CO₂不能供给呼吸但不是有毒气体
- CO还原实验要尾气处理
