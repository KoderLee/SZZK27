## 知识点1：分式的概念与基本性质

### 内容
**分式的概念**：
- 形如A/B的式子，其中A、B是整式，且B中含有字母，B≠0
- A叫做分子，B叫做分母

**分式有意义的条件**：
- 分母不等于0

**分式值为0的条件**：
- 分子等于0且分母不等于0

**分式的基本性质**：
- 分式的分子与分母同乘（或除以）一个不等于0的整式，分式的值不变
- A/B = (A·M)/(B·M) = (A÷M)/(B÷M)（M≠0）

**约分**：
- 把一个分式的分子和分母的公因式约去，这种变形称为分式的约分
- 约分后分子分母没有公因式的分式称为最简分式

## 知识点2：分式的运算

### 内容
**分式的乘除**：
- 乘法：(a/b)·(c/d) = (ac)/(bd)
- 除法：(a/b)÷(c/d) = (a/b)·(d/c) = (ad)/(bc)
- 运算结果要化为最简分式

**分式的加减**：
- 同分母：(a/c) ± (b/c) = (a±b)/c
- 异分母：先通分，再加减
  - (a/b) ± (c/d) = (ad)/(bd) ± (bc)/(bd) = (ad±bc)/(bd)

**通分**：
- 把几个异分母分式化成与原来相等的同分母分式的过程
- 关键是找最简公分母

**最简公分母的确定**：
- 系数：各分母系数的最小公倍数
- 字母：各分母所有字母的最高次幂

**分式的混合运算**：
- 先算乘方，再算乘除，最后算加减
- 有括号的先算括号里面的

## 知识点3：分式方程

### 内容
**分式方程的概念**：
- 分母中含有未知数的方程叫做分式方程

**分式方程的解法**：
1. 去分母：方程两边同乘最简公分母，化为整式方程
2. 解整式方程
3. 检验：将解代入最简公分母
   - 若最简公分母不为0，则是原方程的解
   - 若最简公分母为0，则是增根，原方程无解

**增根**：
- 去分母后产生的使原分式方程分母为0的根

**解分式方程的注意事项**：
- 必须检验，这是解分式方程的必要步骤
- 去分母时不要漏乘不含分母的项

- 增根是使分母为0的值

## 知识点4：分式方程的应用

### 内容
**列分式方程解应用题的一般步骤**：
1. 审：审题，找出等量关系
2. 设：设未知数
3. 列：根据等量关系列分式方程
4. 解：解分式方程
5. 验：检验解是否符合实际意义
6. 答：写出答案

**常见应用类型**：
- 行程问题：路程=速度×时间
- 工程问题：工作量=工作效率×工作时间
- 利润问题：利润=售价-进价
- 配套问题

## 知识点5：整数指数幂

### 内容
**负整数指数幂**：
- a⁻ⁿ = 1/aⁿ（a≠0，n为正整数）
- 特别地：a⁻¹ = 1/a

**零指数幂**：
- a⁰ = 1（a≠0）

**整数指数幂的运算性质**：
- aᵐ·aⁿ = aᵐ⁺ⁿ（m、n为整数）
- (aᵐ)ⁿ = aᵐⁿ（m、n为整数）
- (ab)ⁿ = aⁿbⁿ（n为整数）
- aᵐ÷aⁿ = aᵐ⁻ⁿ（a≠0，m、n为整数）

**科学记数法**：
- 小于1的正数可以表示为a×10⁻ⁿ的形式，其中1≤|a|＜10，n为正整数
- n等于原数左起第一个非零数字前所有零的个数（包括小数点前的零）
