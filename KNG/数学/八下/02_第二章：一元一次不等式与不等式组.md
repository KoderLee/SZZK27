## 知识点1：不等关系与不等式

### 内容
**不等式的概念**：
- 用不等号（<、>、≤、≥、≠）连接的式子叫做不等式

**不等式的基本性质**：
- 性质1：不等式两边加（或减）同一个数（或式子），不等号方向不变
  - 若a > b，则a ± c > b ± c
- 性质2：不等式两边乘（或除以）同一个正数，不等号方向不变
  - 若a > b，c > 0，则ac > bc，a/c > b/c
- 性质3：不等式两边乘（或除以）同一个负数，不等号方向改变
  - 若a > b，c < 0，则ac < bc，a/c < b/c

**列不等式**：
- 根据实际问题中的数量关系列出不等式
- 关键词："大于"、"小于"、"至少"、"至多"、"不超过"、"不少于"等

### 注意事项
- 性质3中不等号方向要改变
- 除以未知数时要注意符号讨论
- "≤"表示"小于或等于"，"≥"表示"大于或等于"

## 知识点2：一元一次不等式

### 内容
**一元一次不等式的概念**：
- 含有一个未知数，未知数的次数是1的不等式

**一元一次不等式的解法**：
- 步骤：去分母 → 去括号 → 移项 → 合并同类项 → 系数化为1
- 注意：系数化为1时，若系数为负数，不等号方向要改变

**一元一次不等式的解集**：
- 使不等式成立的未知数的取值范围
- 解集在数轴上的表示：
  - ">"、"<"：用空心圆圈
  - "≥"、"≤"：用实心圆点
  - 方向：大于向右，小于向左

### 出处
- 教材：八年级下册数学（北师大版）第二章
- 页码：第40-45页

### 中考出题概率
90%

### 常见题型
- 解一元一次不等式
- 在数轴上表示解集
- 求不等式的整数解

### 注意事项
- 去分母时各项都要乘，不能漏乘
- 系数化为1时注意不等号方向
- 数轴表示时注意空心与实心的区别

## 知识点3：一元一次不等式与一次函数

### 内容
**一次函数与一元一次不等式的关系**：
- 一次函数y = kx + b的图象在x轴上方（y > 0）的部分对应kx + b > 0的解集
- 一次函数y = kx + b的图象在x轴下方（y < 0）的部分对应kx + b < 0的解集

**利用函数图象解不等式**：

## 知识点4：一元一次不等式组

### 内容
**一元一次不等式组的概念**：
- 由几个含有同一个未知数的一元一次不等式组成的不等式组

**一元一次不等式组的解法**：
- 分别求出每个不等式的解集
- 找出这些解集的公共部分（即不等式组的解集）

**不等式组解集的确定（口诀）**：
- 同大取大：x > a且x > b（a > b），则x > a
- 同小取小：x < a且x < b（a < b），则x < b
- 大小小大中间找：x > a且x < b（a < b），则a < x < b
- 大大小小找不到：x > b且x < a（a < b），则无解

**解集的表示**：
- 可在数轴上表示各不等式的解集，找出公共部分

- 在数轴上表示不等式组的解集
- 求不等式组的特殊解（整数解、正整数解等）

### 注意事项
- 解每个不等式时独立进行
- 找公共部分时借助数轴更直观
- 无解的情况也要会判断

## 知识点5：一元一次不等式（组）的应用

### 内容
**列不等式（组）解应用题的一般步骤**：
1. 审：审题，找出不等关系
2. 设：设未知数
3. 列：根据不等关系列不等式（组）
4. 解：解不等式（组）
5. 验：检验解是否符合实际意义
6. 答：写出答案

**常见应用类型**：
- 分配问题：至少、至多、不超过
- 方案选择问题：比较不同方案
- 最值问题：最大利润、最少成本
- 范围问题：取值范围的确定

**注意实际问题中的限制**：
- 人数必须为正整数
- 长度、面积等必须为正数
- 时间、价格等的实际意义
