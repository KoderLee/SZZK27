## 知识点1：字母表示数

### 内容
**字母表示数的意义**：
- 可以表示任意数
- 可以表示运算律、公式
- 可以表示数量关系
- 可以表示未知数

**用字母表示运算律**：
- 加法交换律：a + b = b + a
- 加法结合律：(a + b) + c = a + (b + c)
- 乘法交换律：ab = ba
- 乘法结合律：(ab)c = a(bc)
- 乘法分配律：a(b + c) = ab + ac

**用字母表示公式**：
- 长方形面积：S = ab
- 长方形周长：C = 2(a + b)
- 圆的面积：S = πr²
- 圆的周长：C = 2πr
- 三角形面积：S = (1/2)ah
- 路程公式：s = vt

**用字母表示数量关系**：
- 比a大5的数：a + 5
- a的3倍：3a
- a的一半：(1/2)a 或 a/2
- a与b的和的平方：(a + b)²
- a、b两数的平方和：a² + b²

**书写规范**：
- 数字与字母相乘：数字在前，字母在后，乘号省略
  - 例：3 × a = 3a
- 字母与字母相乘：乘号省略或用"·"
  - 例：a × b = ab 或 a·b
- 带分数要化为假分数
  - 例：(5/2)a，不写2(1/2)a
- 除法写成分数形式
  - 例：a ÷ b = a/b
- 单位前有加减运算要加括号
  - 例：(a + b)米

## 知识点2：代数式

### 内容
**代数式的定义**：
用运算符号把数和表示数的字母连接而成的式子叫做代数式。
单独的一个数或一个字母也是代数式。

**代数式的分类（初步）**：
- **有理式**
  - 整式（单项式、多项式）
  - 分式（以后学习）
- **根式**（以后学习）

**列代数式的方法**：
- 抓住关键词：和、差、积、商、大、小、多、少、倍、分
- 理清运算顺序
- 注意括号的使用
- 检验代数式是否正确

**常见数量关系的代数式表示**：
| 数量关系 | 代数式 |
|----------|--------|
| a与b的和 | a + b |
| a与b的差 | a - b |
| a与b的积 | ab |
| a与b的商 | a/b |
| a的相反数 | -a |
| a的倒数 | 1/a (a≠0) |
| a的绝对值 | |a| |
| a的平方 | a² |
| a的立方 | a³ |
| a与b的和的平方 | (a+b)² |
| a与b的平方和 | a²+b² |
| 比a大5的数 | a+5 |
| 比a小3的数 | a-3 |
| a的3倍 | 3a |
| a的一半 | a/2 |

**代数式的值**：
用具体数值代替代数式中的字母，按照运算关系计算得出的结果。

**求代数式的值的步骤**：
1. 代入：用数值代替代数式中的字母
2. 计算：按运算顺序计算结果

**注意事项**：
- 代入负数、分数时要加括号
- 注意运算顺序
- 注意代数式有意义的条件

## 知识点3：整式

### 内容
**单项式**：
- **定义**：由数与字母的积组成的代数式叫做单项式
- 单独的一个数或一个字母也是单项式

**单项式的系数**：
- 单项式中的数字因数叫做这个单项式的系数
- 例：-3x²y的系数是-3

**单项式的次数**：
- 一个单项式中，所有字母的指数的和叫做这个单项式的次数
- 例：3x²y³的次数是2+3=5

**单项式的注意事项**：
- 系数包括前面的符号
- 系数是1或-1时，1通常省略不写
- 单独的数字的次数是0
- π是数字，不是字母

**多项式**：
- **定义**：几个单项式的和叫做多项式
- **项**：组成多项式的每个单项式叫做多项式的项
- **常数项**：不含字母的项叫做常数项

**多项式的次数**：
- 多项式里次数最高项的次数，叫做这个多项式的次数
- 例：3x² + 2x - 1的次数是2

**多项式的命名**：
- 按次数：一次、二次、三次...
- 按项数：二项式、三项式...
- 例：3x² + 2x - 1是二次三项式

**整式**：
- **定义**：单项式和多项式统称为整式
- 特征：分母不含字母

**升幂排列与降幂排列**：
- **升幂排列**：按某一字母的指数从小到大排列
- **降幂排列**：按某一字母的指数从大到小排列

## 知识点4：合并同类项

### 内容
**同类项**：
- **定义**：所含字母相同，并且相同字母的指数也相同的项叫做同类项
- 几个常数项也是同类项

**同类项的判断标准**：
1. 字母相同
2. 相同字母的指数相同
3. 与系数无关
4. 与字母顺序无关

**合并同类项**：
- **定义**：把多项式中的同类项合并成一项
- **法则**：
  - 同类项的系数相加
  - 所得的结果作为系数
  - 字母和字母的指数不变

**合并同类项的步骤**：
1. 找出同类项（可标记）
2. 同类项系数相加
3. 字母和指数不变
4. 写出合并后的结果

**合并同类项的注意事项**：
- 不是同类项不能合并
- 系数相加时注意符号
- 系数为0时，该项消去
- 项数较多时可分组合并

**找同类项的技巧**：
- 相同字母完全相同
- 相同字母的指数对应相等
- 只看字母部分，不看系数

## 知识点5：去括号与添括号

### 内容
**去括号法则**：
- **括号前是"+"号**：把括号和它前面的"+"号去掉后，原括号里各项的符号都不改变
  - 例：+(a - b) = a - b
- **括号前是"-"号**：把括号和它前面的"-"号去掉后，原括号里各项的符号都要改变
  - 例：-(a - b) = -a + b

**去括号的注意事项**：
- 括号前有数字因数时，要先用乘法分配律
  - 例：-2(3a - 2b) = -6a + 4b
- 多层括号从内向外去，或从外向内去
- 去括号后要合并同类项

**添括号法则**：
- **添上"+"号和括号**：括到括号里的各项都不变符号
  - 例：a - b = +(a - b)
- **添上"-"号和括号**：括到括号里的各项都改变符号
  - 例：a - b = -(-a + b)

**添括号的应用**：
- 分组分解
- 凑整计算
- 化简求值

**多重括号的化简**：
- **顺序**：先去小括号，再去中括号，最后去大括号
- **技巧**：逐层去括号，每去一层合并一次

**常见错误**：
- 括号前是负号，去括号后忘记变号
- 括号前有系数，漏乘某些项
- 只改变第一项的符号

## 知识点6：整式的加减

### 内容
**整式加减的实质**：
去括号和合并同类项。

**整式加减的步骤**：
1. **去括号**：按去括号法则去掉括号
2. **找同类项**：识别同类项
3. **合并同类项**：同类项系数相加
4. **写出结果**：按习惯排列（降幂或升幂）

**整式加减的应用**：
- 化简求值
- 列整式表示数量关系
- 证明恒等式
- 解方程的基础

**化简求值的步骤**：
1. 化简：去括号、合并同类项
2. 代入：用数值代替字母
3. 计算：算出结果

**整体代入法**：
- 条件不是单独字母的值，而是代数式的值
- 将所求代数式变形，整体代入计算
- 例：已知x+y=3，求2x+2y+1=2(x+y)+1=7

**整式加减的常见题型**：
1. 直接化简
2. 化简求值
3. 不含某项（系数为0）
4. 与某字母无关
5. 看错符号问题

**注意事项**：
- 去括号要变号（括号前是负号）
- 系数要乘到每一项
- 合并同类项要彻底
- 代入负数、分数要加括号
