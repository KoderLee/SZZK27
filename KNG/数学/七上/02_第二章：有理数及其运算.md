## 知识点1：有理数

### 内容
**正数和负数**：
- **正数**：大于0的数，如3，1.5，+2等（"+"号可省略）
- **负数**：小于0的数，如-3，-1.5，-2等（"-"号不可省略）
- **0**：既不是正数，也不是负数

**用正负数表示相反意义的量**：
- 收入与支出
- 上升与下降
- 向东与向西
- 零上与零下
- 规定其中一种量为正，则另一种量为负

**有理数的定义**：
整数和分数统称为有理数。

**有理数的分类**：

**按定义分类**：
- **整数**：正整数、0、负整数
- **分数**：正分数、负分数

**按正负分类**：
- **正有理数**：正整数、正分数
- **0**
- **负有理数**：负整数、负分数

**数轴**：
- **定义**：规定了原点、正方向和单位长度的直线
- **三要素**：原点、正方向、单位长度
- **画法**：一般取向右为正方向
- **作用**：表示数、比较大小

**相反数**：
- **定义**：只有符号不同的两个数互为相反数
- **性质**：a的相反数是-a；0的相反数是0
- **几何意义**：数轴上关于原点对称的两个点

**多重符号化简**：
- "+"号的个数不影响结果
- "-"号的个数：奇数个为负，偶数个为正
- 规律：负负得正，正负得负

**绝对值**：
- **定义**：数轴上表示数a的点与原点的距离，记作|a|
- **性质**：
  - 正数的绝对值是它本身：|a| = a (a > 0)
  - 负数的绝对值是它的相反数：|a| = -a (a < 0)
  - 0的绝对值是0：|0| = 0
- **非负性**：|a| ≥ 0

**倒数**：
- **定义**：乘积为1的两个数互为倒数
- **性质**：a(a≠0)的倒数是1/a
- **注意**：0没有倒数

## 知识点2：有理数的大小比较

### 内容
**利用数轴比较**：
- 数轴上右边的数总比左边的数大
- 正数 > 0 > 负数

**利用法则比较**：
- **正数与正数**：绝对值大的数大
- **负数与负数**：绝对值大的反而小
- **正数与负数**：正数大于负数
- **与0比较**：正数大于0，负数小于0

**比较有理数大小的步骤**：
1. 先化简各数
2. 分类：正数、0、负数
3. 正数按绝对值从大到小排列
4. 负数按绝对值从小到大排列
5. 按"正数 > 0 > 负数"排列

**绝对值的性质应用**：
- 若|a| = |b|，则a = b或a = -b
- 若|a| + |b| = 0，则a = 0且b = 0
- 若|a| = a，则a ≥ 0
- 若|a| = -a，则a ≤ 0

**特殊值法**：
对于字母表示的数，可代入特殊值进行比较。

## 知识点3：有理数的加法

### 内容
**加法法则**：

**同号两数相加**：
- 取相同的符号
- 并把绝对值相加
- 例：(-3) + (-5) = -(3+5) = -8

**异号两数相加**：
- 取绝对值较大的加数的符号
- 并用较大的绝对值减去较小的绝对值
- 例：(-7) + 4 = -(7-4) = -3

**互为相反数相加**：
- 和为0
- 例：5 + (-5) = 0

**一个数与0相加**：
- 仍得这个数
- 例：(-3) + 0 = -3

**加法运算律**：
- **交换律**：a + b = b + a
- **结合律**：(a + b) + c = a + (b + c)

**加法运算技巧**：
- 同号结合法
- 相反数结合法
- 凑整法
- 分数与小数分别结合

**加法的应用**：
- 求和
- 变化量的计算
- 实际问题的建模

## 知识点4：有理数的减法

### 内容
**减法法则**：
- 减去一个数，等于加上这个数的相反数
- a - b = a + (-b)
- 减法转化为加法进行计算

**减法运算步骤**：
1. 将减号变为加号
2. 将减数变为它的相反数
3. 按加法法则计算

**加减混合运算**：
- 统一成加法运算
- 省略加号和括号（代数和形式）
- 运用运算律简化计算

**代数和**：
- 形如：-3 + 5 - 2 + 7 的式子
- 读法："负3、正5、负2、正7的和"或"负3加5减2加7"

**运算技巧**：
- 同号归类
- 凑整（凑0、凑10、凑100等）
- 相反数抵消
- 分数与小数灵活转化

**减法的应用**：
- 求差
- 温度变化
- 海拔高度差
- 时差计算

## 知识点5：有理数的乘法

### 内容
**乘法法则**：

**两数相乘**：
- 同号得正，异号得负
- 并把绝对值相乘
- 例：(-3) × (-4) = 12；(-3) × 4 = -12

**任何数与0相乘**：
- 都得0

**多个有理数相乘**：
- 几个不为0的数相乘，负因数的个数为偶数时积为正，奇数时积为负
- 几个数相乘，有一个因数为0，积为0

**乘法运算律**：
- **交换律**：ab = ba
- **结合律**：(ab)c = a(bc)
- **分配律**：a(b + c) = ab + ac

**倒数的应用**：
- 除法变乘法
- 分数的约分

**乘法运算技巧**：
- 带分数化假分数
- 小数化分数
- 约分后再相乘
- 灵活运用运算律

## 知识点6：有理数的除法

### 内容
**除法法则**：
- **法则一**：除以一个不等于0的数，等于乘这个数的倒数
  - a ÷ b = a × (1/b) (b ≠ 0)
- **法则二**：两数相除，同号得正，异号得负，并把绝对值相除
  - 0除以任何一个不等于0的数，都得0

**除法运算步骤**：
1. 确定商的符号
2. 除法变乘法（乘以倒数）或直接用绝对值相除
3. 计算结果

**乘除混合运算**：
- 统一成乘法运算
- 确定符号（负因数个数）
- 计算绝对值

**四则混合运算**：
- **运算顺序**：
  1. 先算乘方（以后再学）
  2. 再算乘除
  3. 最后算加减
  4. 有括号的先算括号内的（小→中→大）
- **注意**：同级运算从左到右进行

**运算技巧**：
- 小数化分数
- 带分数化假分数
- 除法变乘法
- 能约分的先约分
