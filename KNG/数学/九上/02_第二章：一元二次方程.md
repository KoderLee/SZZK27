## 知识点1：一元二次方程的概念

### 内容
**一元二次方程的定义**：
- 只含有一个未知数，且未知数的最高次数是2的整式方程
- 一般形式：ax² + bx + c = 0（a ≠ 0）

**一元二次方程的项**：
- ax² 叫做二次项，a 叫做二次项系数
- bx 叫做一次项，b 叫做一次项系数
- c 叫做常数项

**一元二次方程的根**：
- 使方程左右两边相等的未知数的值
- 一元二次方程最多有两个根

**一元二次方程的识别**：
- 必须是整式方程
- 只含有一个未知数
- 未知数的最高次数是2
- 二次项系数不为0

## 知识点2：解一元二次方程——直接开平方法和配方法

### 内容
**直接开平方法**：
- 适用于形如x² = p或(x+m)² = p的方程
- 当p > 0时，方程有两个不等实根：x = ±√p
- 当p = 0时，方程有两个相等实根：x = 0
- 当p < 0时，方程无实根

**配方法**：
- 通过配方把方程化为(x+m)² = p的形式
- 配方步骤：
  1. 二次项系数化为1
  2. 移项：常数项移到右边
  3. 配方：两边加一次项系数一半的平方
  4. 写成完全平方形式
  5. 用直接开平方法求解

**配方法的应用**：
- 解一元二次方程
- 求二次三项式的最值
- 证明代数式的正负性

### 注意事项
- 配方时要保证等式两边同时加
- 配方的关键是加一次项系数一半的平方
- 二次项系数不为1时，先化为1

## 知识点3：解一元二次方程——公式法

### 内容
**求根公式**：
- 对于ax² + bx + c = 0（a ≠ 0），当b² - 4ac ≥ 0时：
- x = [-b ± √(b²-4ac)] / (2a)

**根的判别式**：
- Δ = b² - 4ac
- Δ > 0：方程有两个不相等的实数根
- Δ = 0：方程有两个相等的实数根
- Δ < 0：方程没有实数根

**公式法的步骤**：
1. 将方程化为一般形式，确定a、b、c
2. 计算判别式Δ = b² - 4ac
3. 判断Δ的符号
4. 若Δ ≥ 0，代入求根公式求解

## 知识点4：解一元二次方程——因式分解法

### 内容
**因式分解法**：
- 将方程一边化为0，另一边分解为两个一次因式的积
- 利用若ab = 0，则a = 0或b = 0的原理求解

**常用方法**：
- 提公因式法
- 平方差公式
- 完全平方公式
- 十字相乘法

**因式分解法的步骤**：
1. 将方程右边化为0
2. 将左边分解因式
3. 令每个因式等于0，得到两个一元一次方程
4. 分别求解

**适用情况**：
- 方程一边容易分解因式
- 缺少一次项或常数项的方程

### 注意事项
- 右边必须化为0
- 分解要彻底
- 不能约去含未知数的因式（可能丢根）

3. 列：根据等量关系列方程
4. 解：解方程

## 知识点6：根与系数的关系（韦达定理）

### 内容
**韦达定理**：
- 若x₁、x₂是ax² + bx + c = 0（a ≠ 0）的两根，则：
- x₁ + x₂ = -b/a
- x₁ · x₂ = c/a

**常见变形**：
- x₁² + x₂² = (x₁+x₂)² - 2x₁x₂
- (x₁-x₂)² = (x₁+x₂)² - 4x₁x₂
- 1/x₁ + 1/x₂ = (x₁+x₂)/(x₁x₂)

**应用**：
- 已知一根求另一根
- 不求根计算对称式的值
- 构造以已知数为根的一元二次方程

**逆定理**：
- 以x₁、x₂为根的方程（二次项系数为1）是：
- x² - (x₁+x₂)x + x₁x₂ = 0
