# 物理 - 2027年深圳中考

## 考试信息

| 项目 | 内容 |
|:--|:--|
| 分值 | 与化学合卷共140分（笔试合卷120分 + 实验操作20分），物理约占70分 |
| 考试形式与时长 | 与化学合卷笔试 + 实验操作考试 |
| 关键变化/备注 | 重大变化：实验操作分值由12分增至20分，笔试为合卷（物理+化学） |
