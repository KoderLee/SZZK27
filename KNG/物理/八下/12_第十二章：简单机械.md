## 知识点1：杠杆

### 内容
**杠杆的概念**：
- **定义**：在力的作用下能绕固定点转动的硬棒叫杠杆
- **条件**：硬棒、能转动、有固定点

**杠杆的五要素**：
1. **支点（O）**：杠杆绕着转动的固定点
2. **动力（F₁）**：使杠杆转动的力
3. **阻力（F₂）**：阻碍杠杆转动的力
4. **动力臂（l₁）**：从支点到动力作用线的距离
5. **阻力臂（l₂）**：从支点到阻力作用线的距离

**力臂的画法**：
1. 找支点O
2. 画力的作用线（延长虚线）
3. 从支点向力的作用线作垂线
4. 标垂足，用大括号标力臂

**杠杆的平衡条件**：
- **内容**：动力×动力臂 = 阻力×阻力臂
- **公式**：F₁l₁ = F₂l₂
- **变形**：F₁/F₂ = l₂/l₁

**杠杆的分类**：

| 类型 | 力臂关系 | 力的关系 | 特点 | 实例 |
|------|----------|----------|------|------|
| 省力杠杆 | l₁ > l₂ | F₁ < F₂ | 省力费距离 | 撬棍、钢丝钳、羊角锤 |
| 费力杠杆 | l₁ < l₂ | F₁ > F₂ | 费力省距离 | 镊子、筷子、钓鱼竿 |
| 等臂杠杆 | l₁ = l₂ | F₁ = F₂ | 不省力不费力 | 天平、定滑轮 |

**杠杆最小力的画法**：
- 找离支点最远的点作为力的作用点
- 连接支点和作用点作为力臂（最大力臂）
- 作垂直于力臂的力（最小力）

- 力臂一定垂直于力的作用线
- 杠杆平衡可以是静止或匀速转动
- 最小力对应最大力臂

## 知识点2：滑轮

### 内容
**定滑轮**：
- **定义**：轴固定不动的滑轮
- **实质**：等臂杠杆（动力臂=阻力臂=轮半径）
- **特点**：
  - 不省力（F = G）
  - 不省距离
  - 可以改变力的方向

**动滑轮**：
- **定义**：轴随物体一起运动的滑轮
- **实质**：动力臂是阻力臂2倍的杠杆
- **特点**：
  - 省力（F = G/2，不计滑轮重和摩擦）
  - 费距离（s = 2h）
  - 不能改变力的方向

**滑轮组**：
- **定义**：定滑轮和动滑轮组合在一起
- **省力情况**：
  - F = G/n（n为承担物重的绳子段数，不计滑轮重和摩擦）
  - F = (G + G动)/n（计动滑轮重）
- **距离关系**：s = nh（s为绳端移动距离，h为物体上升高度）
- **速度关系**：v绳 = nv物

**确定绳子段数n的方法**：
- 在动滑轮和定滑轮之间画一条虚线，数与动滑轮相连的绳子段数

**滑轮组的绕法**：
- "奇动偶定"：n为奇数时，绳子固定在动滑轮上；n为偶数时，绳子固定在定滑轮上
- 从少到多绕，从内到外绕

## 知识点3：机械效率

### 内容
**有用功（W有）**：
- 为了达到目的必须做的功
- 竖直提升物体：W有 = Gh
- 水平拉动物体：W有 = fs

**额外功（W额）**：
- 并非我们需要但又不得不做的功
- 克服机械自重、摩擦等做的功

**总功（W总）**：
- 有用功与额外功之和
- W总 = W有 + W额
- 动力做的功：W总 = Fs

**机械效率**：
- **定义**：有用功跟总功的比值叫机械效率
- **公式**：η = W有/W总 × 100%
- **特点**：
  - 机械效率总是小于1（因为W额 > 0）
  - 用百分数表示
  - 无单位

**提高机械效率的方法**：
1. 减小额外功：减小摩擦、减小机械自重
2. 增大有用功：在机械承受范围内增大物重

**常见简单机械的机械效率**：

**杠杆**：
- η = Gh/Fs（s为动力移动距离）
- 主要额外功：转轴处的摩擦

**滑轮组**：
- 竖直提升：η = Gh/Fs = Gh/(Fnh) = G/(nF)
- 不计摩擦和绳重：η = G/(G + G动)

**斜面**：
- η = Gh/Fs（h为斜面高，s为斜面长）
- 主要额外功：克服摩擦做功

- 同一机械的机械效率不是固定不变的
- 计算时注意单位统一

## 知识点4：机械效率综合计算

### 内容
**滑轮组机械效率的几种表达式**：

1. **通用公式**：
   η = W有/W总 = Gh/Fs

2. **已知G、F、n**：
   η = G/(nF)

3. **不计摩擦和绳重**：
   η = G/(G + G动)

4. **由η求G动**：
   G动 = G(1-η)/η

**斜面机械效率**：
- η = Gh/Fs
- 额外功：W额 = fs（克服摩擦力做功）
- 摩擦力的计算：f = (F - Gh/s)

**功、功率、机械效率的综合**：
- W有 = Gh
- W总 = Fs
- P总 = W总/t = Fv
- η = W有/W总

**注意区分**：
| 物理量 | 定义 | 公式 | 单位 | 意义 |
|--------|------|------|------|------|
| 功 | 做功多少 | W = Fs | J | 能量转化多少 |
| 功率 | 做功快慢 | P = W/t | W | 做功快慢 |
| 机械效率 | 有用功占比 | η = W有/W总 | 无 | 机械性能好坏 |
