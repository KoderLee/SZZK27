## 知识点1：浮力

### 内容
**浮力的概念**：
- **定义**：浸在液体（或气体）中的物体受到向上的力叫浮力
- **方向**：竖直向上
- **施力物体**：液体或气体
- **受力物体**：浸在液体或气体中的物体

**浮力产生的原因**：
- 液体对物体上下表面的压力差
- 向上压力 F向上 > 向下压力 F向下
- 浮力 F浮 = F向上 - F向下

**浮力的测量（称重法）**：
- F浮 = G - F拉
  - G：物体在空气中的重力
  - F拉：物体浸在液体中时弹簧测力计的拉力

**决定浮力大小的因素**：
1. **液体的密度**（ρ液）：液体密度越大，浮力越大
2. **物体排开液体的体积**（V排）：排开液体体积越大，浮力越大
3. **与物体密度无关**
4. **与物体浸没深度无关**（完全浸没后）

**阿基米德原理**：
- **内容**：浸在液体中的物体受到向上的浮力，浮力的大小等于物体排开的液体所受的重力
- **公式**：F浮 = G排 = ρ液gV排
  - F浮：浮力（N）
  - G排：排开液体的重力（N）
  - ρ液：液体密度（kg/m³）
  - g：9.8 N/kg（或10 N/kg）
  - V排：排开液体的体积（m³）

**对阿基米德原理的理解**：
- 适用于液体和气体
- V排是排开液体的体积，不是物体体积
  - 完全浸没：V排 = V物
  - 部分浸入：V排 < V物
- 浮力大小只与ρ液和V排有关

## 知识点2：物体的浮沉条件

### 内容
**物体的浮沉条件（受力分析）**：

| 浮沉情况 | 受力关系 | 密度关系（实心物体）| 状态 |
|----------|----------|---------------------|------|
| 上浮 | F浮 > G | ρ物 < ρ液 | 过程 |
| 下沉 | F浮 < G | ρ物 > ρ液 | 过程 |
| 悬浮 | F浮 = G | ρ物 = ρ液 | 状态 |
| 漂浮 | F浮 = G | ρ物 < ρ液 | 状态 |
| 沉底 | F浮 + F支 = G | ρ物 > ρ液 | 状态 |

**对浮沉条件的理解**：
- **上浮**：向上运动的过程，最终变为漂浮
- **下沉**：向下运动的过程，最终变为沉底
- **悬浮**：可以停在液体中任何深度
- **漂浮**：部分露出液面，V排 < V物
- **悬浮与漂浮的区别**：
  - 悬浮：V排 = V物，可停在任何深度
  - 漂浮：V排 < V物，只能在液面

**漂浮条件的重要推论**：
- F浮 = G物
- ρ液gV排 = ρ物gV物
- ρ液V排 = ρ物V物
- V排/V物 = ρ物/ρ液

**改变浮沉的方法**：
1. **改变浮力**：
   - 改变液体密度（如向水中加盐）
   - 改变排开液体体积（如潜水艇的浮沉）
2. **改变重力**：
   - 改变自身重力（如潜水艇吸水排水）

- 漂浮时F浮=G，V排<V物
- 悬浮时F浮=G，V排=V物
- 比较密度时注意是实心物体

## 知识点3：浮力的应用

### 内容
**轮船**：
- **原理**：采用空心办法，增大排开水的体积，增大浮力
- **排水量**：轮船满载时排开水的质量
  - 排水量 = 船自身质量 + 载货质量
- **浮沉**：漂浮，F浮 = G船

**潜水艇**：
- **原理**：通过改变自身重力实现上浮和下潜
- **方法**：吸水（重力增大，下沉）；排水（重力减小，上浮）
- **浮沉**：悬浮或漂浮

**气球和飞艇**：
- **原理**：充入密度小于空气的气体（氢气、氦气、热空气）
- F浮 > G时上升，F浮 < G时下降

**密度计**：
- **原理**：漂浮条件，F浮 = G（不变）
- **特点**：
  - 刻度上小下大
  - 刻度不均匀，上疏下密
  - 读数是被测液体的密度

**浮力计算的方法总结**：

| 方法 | 公式 | 适用情况 |
|------|------|----------|
| 称重法 | F浮 = G - F拉 | 有弹簧测力计读数 |
| 压力差法 | F浮 = F向上 - F向下 | 知道上下表面压力 |
| 阿基米德原理 | F浮 = ρ液gV排 | 知道液体密度和排开体积 |
| 平衡法 | F浮 = G物 | 漂浮或悬浮 |

## 知识点4：浮力计算专题

### 内容
**浮力问题的基本思路**：
1. 判断物体状态（漂浮、悬浮、沉底）
2. 选择合适的方法计算浮力
3. 结合其他物理知识（密度、压强等）求解

**常见浮力问题类型**：

**类型一：判断浮沉状态**
- 比较ρ物和ρ液
- 或比较F浮和G

**类型二：计算浮力大小**
- 称重法：F浮 = G - F拉
- 阿基米德原理：F浮 = ρ液gV排
- 平衡法：F浮 = G（漂浮或悬浮）

**类型三：计算物体密度**
- 漂浮：ρ物 = (V排/V物)ρ液
- 悬浮：ρ物 = ρ液
- 沉底：ρ物 = m/V，需测质量和体积

**类型四：液面变化问题**
- 冰块融化：液面不变（ρ物 < ρ液时）
- 物体放入取出：看V排的变化

**类型五：连接体问题**
- 整体法分析受力
- 隔离法分析单个物体

**浮力与压强的综合**：
- 液体对容器底的压力变化
- 容器对桌面的压力变化
