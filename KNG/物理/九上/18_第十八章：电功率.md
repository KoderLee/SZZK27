## 知识点1：电能 电功

### 内容
**电能（电功W）**：
- **定义**：电流所做的功叫电功，电能转化为其他形式能的过程就是电流做功的过程
- **单位**：焦耳（J），常用千瓦时（kW·h，度）
- **换算**：1 kW·h = 3.6×10⁶ J
- **计算公式**：
  - W = UIt（通用）
  - W = Pt（通用）
  - W = I²Rt（纯电阻）
  - W = U²t/R（纯电阻）

**电能表**：
- **作用**：测量电路消耗的电能
- **读数**：月末示数 - 月初示数 = 本月用电量（kW·h）
- **参数含义**：
  - 220V：额定电压
  - 10(20)A：标定电流10A，最大允许电流20A
  - 600 r/kW·h：每消耗1 kW·h电能，转盘转600转

## 知识点2：电功率

### 内容
**电功率（P）**：
- **定义**：电流做功的快慢，等于电功与时间之比
- **单位**：瓦特（W），常用千瓦（kW）
- **换算**：1 kW = 10³ W
- **计算公式**：
  - P = UI（通用）
  - P = I²R（纯电阻）
  - P = U²/R（纯电阻）

**额定功率与实际功率**：

| 比较项目 | 额定功率 | 实际功率 |
|----------|----------|----------|
| 定义 | 用电器在额定电压下工作的功率 | 用电器在实际电压下工作的功率 |
| 电压 | 额定电压U额 | 实际电压U实 |
| 关系 | P额 = U额I额 | P实 = U实I实 |
| 亮度 | 正常发光 | 电压高则亮，电压低则暗 |

**关系**：
- 当U实 = U额时，P实 = P额，用电器正常工作
- 当U实 > U额时，P实 > P额，用电器可能损坏
- 当U实 < U额时，P实 < P额，用电器不能正常工作

**灯泡亮度**：
- 由实际功率决定
- 实际功率越大，灯泡越亮

## 知识点3：测量小灯泡的电功率

### 内容
**实验原理**：P = UI

**实验器材**：
电源、开关、滑动变阻器、电流表、电压表、小灯泡、导线

**电路图**：
与伏安法测电阻相同

**实验步骤**：
1. 按电路图连接电路
2. 调节滑动变阻器，使电压表示数分别小于、等于、大于额定电压
3. 记录对应的电流值，计算功率
4. 观察灯泡亮度

**注意事项**：
- 连接电路时开关断开
- 滑动变阻器滑片移到阻值最大处
- 电压不能超过额定电压太多（防止烧坏灯泡）
- 不需要求平均值（功率与电压有关）

**实验结论**：
- 实际电压越高，实际功率越大，灯泡越亮
- 小灯泡的电阻随温度升高而增大

## 知识点4：焦耳定律

### 内容
**电流的热效应**：
- 电流通过导体时电能转化为内能的现象

**焦耳定律**：
- **内容**：电流通过导体产生的热量跟电流的平方成正比，跟导体的电阻成正比，跟通电时间成正比
- **公式**：Q = I²Rt
  - Q：热量（J）
  - I：电流（A）
  - R：电阻（Ω）
  - t：时间（s）

**纯电阻电路**：
- 电能全部转化为内能
- Q = W = UIt = I²Rt = U²t/R = Pt

**非纯电阻电路**（如电动机）：
- 电能主要转化为机械能，少部分转化为内能
- W = UIt（总电功）
- Q = I²Rt（产生的热量）
- W > Q

**电热器**：
- 利用电流热效应工作的设备
- 发热体：电阻率大、熔点高的合金（如镍铬合金）
