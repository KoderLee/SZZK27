## 知识点2：欧姆定律

### 内容
**欧姆定律**：
- **内容**：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比
- **公式**：I = U/R
  - I：电流（A）
  - U：电压（V）
  - R：电阻（Ω）
- **变形公式**：
  - U = IR（求电压）
  - R = U/I（求电阻）

**对欧姆定律的理解**：
- 同一性：I、U、R对应同一导体或同一段电路
- 同时性：I、U、R是同一时刻的值
- 统一性：单位要统一（A、V、Ω）

**电阻的计算**：
- R = U/I是计算式，不是决定式
- 电阻是导体本身属性，与U、I无关

## 知识点3：电阻的测量

### 内容
**伏安法测电阻**：
- **原理**：R = U/I（欧姆定律变形）
- **电路图**：电流表串联，电压表并联在被测电阻两端
- **器材**：电源、开关、滑动变阻器、电流表、电压表、待测电阻、导线
- **步骤**：
  1. 按电路图连接电路
  2. 闭合开关，调节滑动变阻器，记录多组U、I值
  3. 计算R = U/I，取平均值
- **滑动变阻器作用**：
  1. 保护电路
  2. 改变待测电阻两端电压，进行多次测量

**减小误差的方法**：
- 多次测量取平均值
- 电表选择合适的量程
- 接线良好，减少接触电阻

## 知识点4：欧姆定律在串并联电路中的应用

### 内容
**串联电路特点**：
- 电流：I = I₁ = I₂ = ...
- 电压：U = U₁ + U₂ + ...
- 电阻：R = R₁ + R₂ + ...
- 分压规律：U₁/U₂ = R₁/R₂（电压与电阻成正比）

**并联电路特点**：
- 电流：I = I₁ + I₂ + ...
- 电压：U = U₁ = U₂ = ...
- 电阻：1/R = 1/R₁ + 1/R₂ + ...
- 分流规律：I₁/I₂ = R₂/R₁（电流与电阻成反比）

**串并联电阻特点**：
- 串联总电阻比任何一个分电阻都大
- 并联总电阻比任何一个分电阻都小
- n个相同电阻R串联：R总 = nR
- n个相同电阻R并联：R总 = R/n
