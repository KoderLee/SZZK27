## 知识点1：鲁迅《从百草园到三味书屋》

### 内容
**作者介绍**：鲁迅（1881-1936），原名周树人，字豫才，浙江绍兴人。中国现代伟大的文学家、思想家、革命家。代表作有小说集《呐喊》《彷徨》《故事新编》，散文集《朝花夕拾》，散文诗集《野草》等。

**作品出处**：《朝花夕拾》（原名《旧事重提》），是鲁迅回忆童年、少年和青年时期不同生活经历与体验的散文集。

**文章主旨**：通过对百草园自由快乐生活和三味书屋学习生活的回忆，表现了儿童热爱大自然、追求自由快乐生活的心理，同时也流露出对封建教育制度一定程度上的批判。

**结构层次**：
- 第一部分（第1-8段）：百草园的快乐生活
  - 百草园的自然景物（第2段）
  - 美女蛇的传说（第3-6段）
  - 冬天雪地捕鸟（第7-8段）
- 第二部分（第9-24段）：三味书屋的读书生活
  - 告别百草园（第9段）
  - 书屋环境（第10-11段）
  - 读书生活（第12-16段）
  - 课下活动（第17-23段）
  - 画画儿（第24段）

**百草园景物描写（第2段）**：
```
不必说碧绿的菜畦，光滑的石井栏，高大的皂荚树，紫红的桑椹；
也不必说鸣蝉在树叶里长吟，肥胖的黄蜂伏在菜花上，轻捷的叫天子（云雀）忽然从草间直窜向云霄里去了。
单是周围的短短的泥墙根一带，就有无限趣味。
```

**写景顺序与角度**：
- 由低到高：菜畦→石井栏→皂荚树→桑椹
- 由高到低：鸣蝉→黄蜂→叫天子
- 由整体到局部：百草园整体→泥墙根一带
- 感官角度：视觉（碧绿、紫红）、听觉（长吟）、触觉（光滑）

**重点语句赏析**：
1. "不必说……也不必说……单是……就……"
   - 句式分析：两个"不必说"概括整体，"单是"突出局部，由整体到局部
   - 表达效果：强调百草园到处都有趣味，处处皆美

2. "油蛉在这里低唱，蟋蟀们在这里弹琴。"
   - 修辞手法：拟人
   - 表达效果：赋予昆虫以人的动作，生动形象地写出百草园的生机与趣味，表现儿童的愉悦心情

3. "Ade，我的蟋蟀们！Ade，我的覆盆子们和木莲们！"
   - 语言特点：德语"Ade"（再见），称呼蟋蟀为"们"，使用第二人称
   - 表达效果：俏皮可爱，表现儿童的天真情趣和对百草园的不舍

4. "先生读书入神的时候……用功。"
   - 细节描写：先生读书的神态、动作
   - 表达效果：生动刻画了私塾先生迂腐而可爱的形象

**写作特色**：
- 写景生动：调动多种感官，运用多种修辞
- 对比鲜明：百草园的快乐与书屋的枯燥对比（但又非完全否定书屋）
- 儿童视角：以儿童的眼光观察世界，充满童真童趣
- 情景交融：在景物描写中融入情感

**重点词语**：
- 确凿：确实
- 菜畦：菜地
- 桑葚：桑树的果实
- 轻捷：轻快敏捷
- 臃肿：过度肥胖，转动不灵
- 相宜：适宜
- 方正：正派
- 博学：学问丰富
- 蝉蜕：蝉的幼虫变为成虫时脱下的壳
- 人声鼎沸：形容人声喧闹，像水在锅里沸腾一样
- 高枕而卧：垫高枕头睡觉，形容不加警惕

### 常见题型
1. **默写题**：默写第2段（百草园景物描写）
2. **写景顺序**：分析第2段的写景顺序和角度
3. **修辞赏析**：分析拟人、排比等修辞的作用
4. **词语理解**：解释"人声鼎沸""高枕而卧"等成语
5. **内容理解**：概括百草园的乐趣；分析书屋生活的特点

## 知识点2：海伦·凯勒《再塑生命的人》

### 内容
**作者介绍**：海伦·凯勒（1880-1968），美国女作家、教育家、慈善家。19个月大时因病失明失聪，在莎莉文老师的帮助下学会阅读、写作和说话。代表作《假如给我三天光明》。

**写作背景**：本文选自《假如给我三天光明》，回忆莎莉文老师来到自己身边后，教自己认识事物、学习知识的经历，表达了对莎莉文老师的感激之情。

**文章主旨**：通过记叙莎莉文老师教"我"认识具体事物（如"水"）的过程，表现了莎莉文老师高超的教育艺术和爱心，表达了"我"对莎莉文老师的感激与敬爱之情。

**结构层次**：
- 第一部分（第1-5段）：莎莉文老师到来前"我"的生活状态（愤怒、痛苦、迷茫）
- 第二部分（第6-13段）：莎莉文老师教"我"认识事物
  - 第6-8段：老师送布娃娃，教拼写"doll"
  - 第9-12段：老师带"我"到井房，教"我"认识"水"
  - 第13段："我"的觉醒与变化

**重点语句赏析**：
1. "我实在有些不耐烦了，抓起新布娃娃就往地上摔，把它摔碎了，心中觉得特别痛快。"
   - 动作描写："抓""摔"表现"我"的愤怒和烦躁
   - 心理描写："觉得特别痛快"表现"我"内心的痛苦
   - 作用：与后文形成对比，突出莎莉文老师的教育成果

2. "水唤醒了我的灵魂，并给予我光明、希望、快乐和自由。"
   - 关键词："唤醒""光明""希望""快乐""自由"
   - 含义：认识"水"开启了海伦认识世界的大门，让她从黑暗中走出来
   - 情感：对莎莉文老师的深深感激

3. "啊！世界上还有比我更幸福的孩子吗？"
   - 句式：反问句
   - 情感：获得知识后的喜悦和幸福，对老师的感激

**人物形象**：
- 莎莉文老师：
  - 富有爱心和耐心：面对暴躁的海伦，始终温柔引导
  - 教育艺术高超：通过实践（让海伦感受水）进行教学
  - 因材施教：根据海伦的特点选择合适的教育方式
- 海伦·凯勒：
  - 求知欲强：一旦开启智慧大门，便如饥似渴学习
  - 聪慧敏感：能敏锐感受老师的爱

**写作特色**：
- 对比手法：学习前后的对比，突出教育的力量
- 心理描写：细腻的心理变化描写
- 第一人称叙述：真实感人，增强感染力

**重点词语**：
- 截然不同：形容两件事物毫无共同之处
- 感慨：有所感触而慨叹
- 疲惫不堪：形容非常疲乏
- 小心翼翼：形容举动十分谨慎，丝毫不敢疏忽
- 不求甚解：现多指只求懂得个大概，不求深刻了解
- 恍然大悟：形容一下子明白过来
- 花团锦簇：形容五彩缤纷、十分华丽的形象
- 美不胜收：美好的东西太多，一时接受不完（看不过来）

## 知识点3：《论语》十二章

### 文学常识
**孔子**（前551-前479），名丘，字仲尼，春秋末期鲁国陬邑（今山东曲阜）人。中国古代伟大的思想家、教育家，儒家学派的创始人。

**《论语》**：儒家经典著作，是记录孔子及其弟子言行的一部书，共20篇。宋代朱熹将其与《大学》《中庸》《孟子》合称为"四书"。

**语录体**：《论语》的文体，记录孔子及其弟子的言行，语言简练，含义深远。

### 原文与翻译

**第一章：学而时习之**
```
子曰："学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？"
```
- 翻译：孔子说："学习了然后按时温习，不是很愉快吗？有志同道合的人从远方来，不是很快乐吗？人家不了解我，我却不恼怒，不是有才德的人吗？"
- 重点词：
  - 子：古代对男子的尊称，这里指孔子
  - 时习：按时温习
  - 说：同"悦"，愉快
  - 愠：生气，恼怒
  - 君子：指有才德的人

**第二章：吾日三省吾身**
```
曾子曰："吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？"
```
- 翻译：曾子说："我每天多次反省自己：替别人办事是不是尽心竭力了呢？同朋友交往是不是诚实可信了呢？老师传授的知识是不是复习了呢？"
- 重点词：
  - 曾子：曾参，孔子的弟子
  - 三省：多次反省。三，泛指多次
  - 忠：尽心竭力
  - 信：诚信
  - 传：传授，这里指老师传授的知识

**第三章：吾十有五而志于学**
```
子曰："吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲，不逾矩。"
```
- 翻译：孔子说："我十五岁开始有志于做学问，三十岁能有所成就，四十岁心里不再感到迷惑，五十岁知道上天的意旨，六十岁能听得进不同的意见，七十岁可以随心所欲，却又不越过法度。"
- 重点词：
  - 有：同"又"，用于整数和零数之间
  - 立：立身，指有所成就
  - 惑：迷惑，疑惑
  - 天命：上天的意旨
  - 耳顺：能听得进不同的意见
  - 逾矩：越过法度

**第四章：温故而知新**
```
子曰："温故而知新，可以为师矣。"
```
- 翻译：孔子说："温习学过的知识，可以得到新的理解和体会，就可以凭借这一点做老师了。"
- 重点词：
  - 温：温习
  - 故：旧的知识
  - 新：新的理解和体会
  - 可以：可以凭借

**第五章：学而不思则罔**
```
子曰："学而不思则罔，思而不学则殆。"
```
- 翻译：孔子说："只是读书却不认真思考，就会迷惑；只空想却不读书，就会疑惑。"
- 重点词：
  - 罔：迷惑，意思是感到迷茫而无所适从
  - 殆：疑惑

**第六章：贤哉回也**
```
子曰："贤哉，回也！一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐。贤哉，回也！"
```
- 翻译：孔子说："颜回的品质多么高尚啊！一竹篮饭，一瓢水，住在简陋的小巷子里，别人都不能忍受这种穷困清苦，颜回却不改变他好学的乐趣。颜回的品质多么高尚啊！"
- 重点词：
  - 箪：古代盛饭用的圆形竹器
  - 堪：能忍受
  - 回：颜回，孔子的弟子

**第七章：知之者不如好之者**
```
子曰："知之者不如好之者，好之者不如乐之者。"
```
- 翻译：孔子说："懂得某种学问的人不如喜爱它的人，喜爱它的人不如把研究这种学问作为快乐的人。"
- 重点词：
  - 知之者：懂得它的人
  - 好之者：喜爱它的人
  - 乐之者：以它为乐的人

**第八章：饭疏食饮水**
```
子曰："饭疏食，饮水，曲肱而枕之，乐亦在其中矣。不义而富且贵，于我如浮云。"
```
- 翻译：孔子说："吃粗粮，喝冷水，弯着胳膊当枕头，乐趣也就在其中了。用不正当的手段得来的富贵，对于我来讲就像是天上的浮云一样。"
- 重点词：
  - 饭：吃
  - 疏食：粗粮
  - 水：冷水
  - 肱：胳膊
  - 不义：不正当的手段

**第九章：三人行**
```
子曰："三人行，必有我师焉。择其善者而从之，其不善者而改之。"
```
- 翻译：孔子说："几个人一起走路，其中必定有可以做我老师的人。选取他们的优点而学习，如果也有他们的缺点就加以改正。"
- 重点词：
  - 三：泛指多个
  - 焉：于此，意思是在其中
  - 善者：优点
  - 从：学习

**第十章：逝者如斯夫**
```
子在川上曰："逝者如斯夫，不舍昼夜。"
```
- 翻译：孔子在河边感叹道："时光像河水一样流去，日夜不停。"
- 重点词：
  - 川：河流
  - 逝者：往者，这里指时光
  - 斯：这，指河水
  - 舍：停

**第十一章：三军可夺帅也**
```
子曰："三军可夺帅也，匹夫不可夺志也。"
```
- 翻译：孔子说："一国军队，可以改变其主帅；一个人的志向却是不能改变的。"
- 重点词：
  - 三军：指军队
  - 夺：改变
  - 匹夫：指平民中的男子，这里泛指平民百姓

**第十二章：博学而笃志**
```
子夏曰："博学而笃志，切问而近思，仁在其中矣。"
```
- 翻译：子夏说："博览群书广泛学习，而且能坚守自己的志向，恳切地提问，多考虑当前的事，仁德就在其中了。"
- 重点词：
  - 子夏：卜商，孔子的弟子
  - 笃：坚守
  - 切问：恳切地提问
  - 近思：多思考当前的事情
  - 仁：仁德

**学习方法**：
- 学而时习之，不亦说乎？
- 温故而知新，可以为师矣。
- 学而不思则罔，思而不学则殆。

**学习态度**：
- 知之者不如好之者，好之者不如乐之者。
- 三人行，必有我师焉。择其善者而从之，其不善者而改之。

## 知识点4：单元写作——写人要抓住特点

### 内容
**写作要求**：
1. 学会观察人物，抓住人物特点
2. 运用细节描写刻画人物
3. 选取典型事例表现人物

**写作技巧**：
1. **观察人物特点**：
   - 外貌特点：容貌、衣着、体态、神情
   - 语言特点：口头禅、说话方式、语气
   - 动作特点：习惯性动作、特殊动作
   - 心理特点：性格、情感、品质

2. **选取典型事例**：
   - 最能表现人物特点的事例
   - 日常生活中的小事
   - 有细节、有画面感

3. **运用描写方法**：
   - 外貌描写：抓住特征，避免面面俱到
   - 语言描写：符合身份，体现性格
   - 动作描写：准确生动，连贯自然
   - 心理描写：真实细腻，符合情境
   - 神态描写：细微之处见精神

4. **描写与叙述结合**：
   - 叙述交代背景、过程
   - 描写刻画细节、渲染气氛

## 知识点6：课外古诗词诵读

### 6.1 《峨眉山月歌》李白

**原文**：
```
峨眉山月半轮秋，影入平羌江水流。
夜发清溪向三峡，思君不见下渝州。
```

**翻译**：
峨眉山上挂着半轮秋月，月影倒映在平羌江水中，随波流淌。夜里我从清溪出发奔向三峡，思念你却难以相见，只能顺流而下到渝州。

**主题**：通过描写峨眉山月夜景，抒发了对友人（或故乡）的思念之情。

**艺术特色**：
- 连用五个地名：峨眉山、平羌江、清溪、三峡、渝州
- 动静结合：山月静，江水流
- 情景交融：在景物描写中融入思乡之情

**重点词语**：
- 半轮秋：半圆的秋月
- 平羌：即青衣江，在峨眉山东北
- 发：出发
- 清溪：即清溪驿，在今四川犍为
- 君：指峨眉山月，也指友人
- 渝州：今重庆一带

**中考出题概率**：75%

### 6.2 《江南逢李龟年》杜甫

**原文**：
```
岐王宅里寻常见，崔九堂前几度闻。
正是江南好风景，落花时节又逢君。
```

**翻译**：
当年在岐王宅里经常见到您的演出，在崔九堂前也曾多次欣赏您的艺术。如今正是江南的大好风景，在落花时节又重逢了您。

**创作背景**：
李龟年是唐代著名音乐家，开元、天宝年间深受玄宗赏识。安史之乱后流落江南。大历五年（770年），杜甫在湖南潭州（今长沙）与李龟年重逢，感时伤世而作此诗。

**主题**：通过回忆往昔与李龟年的交往，抒发了对开元盛世的怀念和对国运衰微、人生飘零的感慨。

**艺术特色**：
- 今昔对比：昔日繁华 vs 今日飘零
- 含蓄蕴藉："落花时节"既是实景，又象征国运衰微、人生暮年
- 言有尽而意无穷

**重点词语**：
- 李龟年：唐代著名音乐家
- 岐王：唐玄宗李隆基的弟弟李范
- 崔九：崔涤，排行第九，唐玄宗宠臣
- 寻常：经常
- 几度：多次
- 落花时节：暮春，也喻指国运衰微、人生暮年

**中考出题概率**：80%

---

### 6.3 《行军九日思长安故园》岑参

**原文**：
```
强欲登高去，无人送酒来。
遥怜故园菊，应傍战场开。
```

**翻译**：
勉强想要按照重阳节的习俗去登高，却没有人像王弘那样送酒来。怜惜远方的故园菊花，应该依傍着战场开放了。

**创作背景**：
安史之乱中，长安沦陷，岑参随军行军，重阳节时思念故园而作。

**主题**：通过重阳登高的联想，表现了诗人对故园的思念和对国事的忧虑。

**艺术特色**：
- 用典："送酒"用陶渊明重阳节饮酒典故
- 想象：想象故园菊花在战场旁开放
- 以小见大：通过菊花寄托忧国忧民之情

**重点词语**：
- 九日：九月九日重阳节
- 强：勉强
- 登高：重阳节习俗
- 送酒：据《南史·隐逸传》记载，陶渊明重阳节无酒，江州刺史王弘派人为他送酒
- 遥怜：远方怜惜
- 傍：靠近

**中考出题概率**：70%

### 6.4 《夜上受降城闻笛》李益

**原文**：
```
回乐烽前沙似雪，受降城外月如霜。
不知何处吹芦管，一夜征人尽望乡。
```

**翻译**：
回乐烽前的沙地白得像雪，受降城外的月色冷得如霜。不知从哪里传来芦笛声，一夜之间出征的将士都遥望故乡。

**主题**：通过描写边塞月夜的景色和芦笛声，抒发了戍边将士的思乡之情。

**艺术特色**：
- 比喻精妙：沙似雪，月如霜
- 视听结合：视觉（沙、月）与听觉（芦管）
- 情景交融：凄清的景色烘托思乡之情
- 以小见大：个人闻笛到"尽望乡"

**重点词语**：
- 受降城：唐代边塞城池，在今内蒙古境内
- 回乐烽：烽火台名
- 芦管：笛子
- 征人：出征的将士
- 尽：都

**中考出题概率**：75%

### 高频易错点
1. "说"通"悦"，不读shuo，读yue
2. "有"通"又"，用于整数和零数之间
3. "愠"是"恼怒"的意思，不是"温暖"
4. "可以"古义是"可以凭借"，不是"能够"
5. "三"在"三省吾身""三人行"中是泛指，不是确数
6. "夺"是"改变"的意思，不是"抢夺"
