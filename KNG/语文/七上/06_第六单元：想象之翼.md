## 知识点1：安徒生《皇帝的新装》

### 内容
**作者介绍**：安徒生（1805-1875），丹麦著名童话作家。世界文学童话创始人。代表作有《卖火柴的小女孩》《海的女儿》《丑小鸭》等。

**作品信息**：本文是安徒生童话中的名篇，写于1837年。

**文章主旨**：通过皇帝被两个骗子愚弄，穿上根本不存在的"新装"赤身裸体举行游行大典的故事，揭露和讽刺了封建统治者的虚伪、愚蠢和自欺欺人的丑恶本质，同时赞扬了孩子的天真无邪和敢于说真话的勇气。

**结构层次**：
- 第一部分（第1段）：皇帝爱新装（故事引子）
- 第二部分（第2-4段）：骗子做新装（故事开端）
- 第三部分（第5-22段）：群臣看新装（故事发展）
  - 老大臣看（第5-14段）
  - 官员看（第15-18段）
  - 皇帝看（第19-22段）
- 第四部分（第23-32段）：游行穿新装（故事高潮）
- 第五部分（第33-36段）：揭穿假新装（故事结局）

**人物形象**：
- 皇帝：
  - 特点：爱慕虚荣、昏庸无能、愚蠢虚伪、自欺欺人
  - 表现：爱新衣成癖；被骗子愚弄；明明看不见却装作看见
- 骗子：
  - 特点：狡猾、善于抓住人心
  - 手段：利用皇帝爱新衣的心理；设置"愚蠢或不称职"的陷阱
- 老大臣、官员：
  - 特点：虚伪、阿谀奉承、明哲保身
  - 表现：明明看不见却说谎；怕被人说愚蠢
- 小孩：
  - 特点：天真无邪、无私无畏、敢于说真话
  - 作用：揭穿谎言，寄托作者理想

**写作特色**：
- 夸张手法：皇帝爱新衣到不理朝政的程度
- 想象丰富：根本不存在的"神奇布料"
- 对比鲜明：成人世界的虚伪与孩子的天真对比
- 讽刺辛辣：对封建统治者的无情讽刺
- 反复手法：多次描写"看不见布料"的情节

**重点语句赏析**：
1. "任何不称职的或者愚蠢得不可救药的人，都看不见这衣服。"
   - 骗子的陷阱，抓住人们怕被人说愚蠢的心理
   - 推动情节发展的关键

2. "难道我是一个愚蠢的人吗？难道我不够资格当一个皇帝吗？这可是我所遇见的一件最可怕的事情。"
   - 心理描写，表现皇帝的虚伪和恐惧
   - 揭示成人世界的心理弱点

3. "可是他什么衣服也没穿呀！"
   - 孩子的话揭穿谎言
   - 与成人的虚伪形成鲜明对比

**重点词语**：
- 炫耀：夸耀
- 愚蠢：愚笨；不聪明
- 滑稽：（言语、动作）引人发笑
- 陛（bì）下：对君主的尊称
- 头衔：指官衔、学衔等称号
- 勋（xūn）章：授给对国家有贡献的人的一种表示荣誉的证章
- 不可救药：病重到已无法救治，比喻人或事物坏到无法挽救的地步
- 骇（hài）人听闻：使人听了非常吃惊（多指社会上发生的坏事）
- 随声附和（hè）：别人说什么，自己跟着说什么，形容没有主见

**主题思想**：
1. 讽刺封建统治者的虚伪、愚蠢
2. 批判成人世界的虚伪和自欺欺人
3. 赞扬孩子的天真无邪和敢于说真话的勇气
4. 呼唤真诚、纯真的社会风气

### 常见题型
1. **内容理解**：概括故事情节；分析人物形象
2. **写作手法**：分析夸张、想象、对比、讽刺的作用
3. **人物分析**：分析皇帝、骗子、大臣、孩子的形象
4. **语句赏析**：赏析关键句的含义
5. **主旨理解**：分析文章的主题思想
6. **拓展延伸**：结合实际谈如何保持童真

### 注意事项
- 理解童话的夸张和想象手法
- 分析人物形象要抓住具体表现
- 理解孩子的作用（揭穿谎言、寄托理想）
- 注意"陛""勋""骇""和"（hè）等易错字
- 理解文章的讽刺意义

## 知识点2：郭沫若《天上的街市》

### 内容
**作者介绍**：郭沫若（1892-1978），原名郭开贞，四川乐山人。现代著名作家、诗人、历史学家、考古学家、古文字学家、社会活动家。代表作有诗集《女神》《星空》，历史剧《屈原》《虎符》等。

**写作背景**：本诗写于1921年，当时中国正处于北洋军阀混战时期，社会黑暗，人民生活困苦。诗人通过对天上街市的想象，表达了对美好生活的向往。

**文章主旨**：通过描绘天上的美丽街市和牛郎织女的幸福生活，表达了对黑暗现实的不满，对理想生活的向往和追求。

**诗歌内容**：
```
远远的街灯明了，
好像闪着无数的明星。
天上的明星现了，
好像点着无数的街灯。

我想那缥缈的空中，
定然有美丽的街市。
街市上陈列的一些物品，
定然是世上没有的珍奇。

你看，那浅浅的天河，
定然是不甚宽广。
那隔着河的牛郎织女，
定能够骑着牛儿来往。

我想他们此刻，
定然在天街闲游。
不信，请看那朵流星，
是他们提着灯笼在走。
```

**结构层次**：
- 第一节：街灯与明星互喻，引发联想
- 第二节：想象天上的美丽街市
- 第三节：想象牛郎织女自由来往
- 第四节：想象他们在天街闲游

**写作特色**：
- 联想与想象：由街灯联想到明星，再想象天街景象
- 反复手法："定然""定能够"反复出现，表达坚定信念
- 化用典故：改写牛郎织女的故事（悲剧→喜剧）
- 语言清新：语言优美，节奏明快

**重点词语**：
- 缥缈：形容隐隐约约，若有若无
- 定然：必定，一定
- 陈列：把物品摆出来供人看
- 珍奇：珍贵而奇异
- 天河：银河
- 不甚：不很
- 闲游：悠闲地游玩
- 灯笼：一种照明用具

**重点语句赏析**：
1. "远远的街灯明了，好像闪着无数的明星。天上的明星现了，好像点着无数的街灯。"
   - 互喻手法：街灯↔明星
   - 回环往复，营造优美意境
   - 为下文想象做铺垫

2. "定然有美丽的街市""定然是世上没有的珍奇""定然是不甚宽广""定能够骑着牛儿来往"
   - "定然""定能够"反复：表达坚定信念
   - 强调这是诗人确信的美好世界

3. "不信，请看那朵流星，是他们提着灯笼在走。"
   - 想象奇特：流星=灯笼
   - 浪漫温馨的画面

**与传说对比**：
- 传统传说：牛郎织女被天河阻隔，每年七夕相会一次（悲剧）
- 诗中故事：天河浅浅，牛郎织女自由来往（喜剧）
- 对比意义：表达对封建束缚的反抗，对自由幸福的向往

### 常见题型
1. **默写题**：全诗默写或重点句默写
2. **写作手法**：分析联想、想象的作用
3. **词语理解**：解释"缥缈""定然""珍奇"等词语
4. **内容理解**：概括诗歌描绘的画面
5. **情感主旨**：分析诗人表达的情感

## 知识点3：女娲造人

### 内容
**作者介绍**：袁珂（1916-2001），四川新都人。神话学家。代表作有《中国古代神话》《山海经校注》等。

**体裁**：神话，是远古时代人民集体口头创作的关于神的故事，是他们对自然现象和社会生活的原始解释。

**文章主旨**：通过女娲造人的故事，表现了人类对自身起源的好奇和探索，歌颂了女娲的勤劳、智慧和伟大的母爱。

**故事内容**：
- 天地开辟后，女娲感到孤独
- 女娲用黄泥造人（一个一个捏）
- 工作量太大，女娲用藤条甩泥造人（批量生产）
- 女娲让男人和女人婚配，自己繁衍后代
- 女娲把人造好后，分为贵贱（神话中的阶级观念）

**女娲形象**：
- 勤劳：不辞辛劳地造人
- 智慧：想出用藤条甩泥的办法
- 有爱心：让人类自己繁衍，不再孤独
- 神通广大：有神力造人

**写作特色**：
- 想象丰富：女娲造人的具体过程
- 细节描写：女娲的心理、动作描写
- 人性化：赋予女娲人的情感和性格
- 神话色彩：充满神奇想象

**重点词语**：
- 女娲：中国神话中的创世女神
- 寂寞：孤单冷清
- 神通：原是佛教用语，指无所不能的力量，后指特别高明的本领
- 莽莽榛榛：形容草木丛生的样子
- 澄澈：清澈透明
- 掘：挖
- 掺和：掺杂混合在一起
- 气概：在对待重大问题上表现的态度、举动或气势（专指正直、豪迈的）
- 灵敏：反应快；能对极其微弱的刺激迅速反应
- 绵延：延续不断

## 知识点4：《寓言四则》

### 文学常识
**寓言**：用假托的故事或自然物的拟人手法来说明某个道理或教训的文学作品，常带有讽刺或劝诫的性质。特点是篇幅短小，寓意深刻。

### 4.1 《赫尔墨斯和雕像者》

#### 内容
**出处**：《伊索寓言》

**故事内容**：
赫尔墨斯（宙斯和赫拉的儿子，掌管旅行和商业的神）想知道他在人间受到多大的尊重，化作凡人来到雕像者的店里。他看见宙斯的雕像，问道值多少钱。雕像者说一个银元。他又笑着问赫拉的雕像值多少钱。雕像者说还要贵一点。后来赫尔墨斯看见自己的雕像，心想自己身为神使，又是商人的庇护神，人们会对他更尊重些，于是问道这个值多少钱。雕像者回答说："假如你买了那两个，这个算添头，白送。"

**寓意**：讽刺那些爱慕虚荣而不被人重视的人。告诫人们要有自知之明，不要妄自尊大。

**人物形象**：
- 赫尔墨斯：爱慕虚荣、自命不凡
- 雕像者：朴实、直率

**写作特色**：
- 对话为主：通过对话展开情节
- 心理描写：赫尔墨斯的心理活动
- 讽刺手法：通过结局讽刺赫尔墨斯

**重点词语**：
- 赫（hè）尔墨斯：古希腊神话中的神
- 雕像者：雕刻人像的人
- 庇护：袒护；保护
- 添头：指买东西时商家为促销而额外赠送的小件物品

### 4.2 《蚊子和狮子》

#### 内容
**出处**：《伊索寓言》

**故事内容**：
蚊子飞到狮子面前，对他说："我不怕你，你并不比我强。若说不是这样，你到底有什么力量呢？是用爪子抓，牙齿咬吗？女人同男人打架，也会这么干。我比你强得多。你要是愿意，我们来较量较量吧！"蚊子吹着喇叭冲过去，专咬狮子鼻子周围没有毛的地方。狮子气得用爪子把自己的脸都抓破了。蚊子战胜了狮子，又吹起喇叭，唱着凯歌飞走，却被蜘蛛网粘住了。蚊子将要被吃掉时，叹息说，自己同最强大的动物都较量过，不料被这小小的蜘蛛消灭了。

**寓意**：讽刺那些打败过大人物却被小人物打败的人。告诫人们：即使取得胜利也不能骄傲自满，要时刻保持警惕。

**写作特色**：
- 拟人手法：赋予蚊子和狮子人的语言和性格
- 对比手法：蚊子与狮子的强弱对比；蚊子战胜狮子与被蜘蛛消灭对比
- 讽刺意味：骄傲自满者的下场

**重点词语**：
- 较量：用竞赛或斗争的方式比本领、实力的高低
- 凯歌：打了胜仗所唱的歌
- 叹息：叹气
- 蜘蛛：一种节肢动物

### 4.3 《穿井得一人》

#### 内容
**出处**：《吕氏春秋·慎行论·察传》

**原文**：
```
宋之丁氏，家无井而出溉汲，常一人居外。及其家穿井，告人曰："吾穿井得一人。"有闻而传之者曰："丁氏穿井得一人。"国人道之，闻之于宋君。宋君令人问之于丁氏，丁氏对曰："得一人之使，非得一人于井中也。"
```

**翻译**：
宋国有户姓丁的人家，家里没有井，需要出门到远处去打水浇田，常常有一人停留在外面（专管打水）。等到他家打了井，便告诉别人说："我家打井得到一个人。"有人听了这话，传播说："丁家挖井挖出了一个人。"都城的人都谈论这件事，一直传到宋国国君那里。宋国国君派人去问姓丁的，丁家的人回答说："得到一个人的劳力，并非从井中挖出一个人来呀。"

**寓意**：告诫人们：对于传闻，要以审慎的态度进行分析、甄别，不要轻易相信传闻，也不要轻易传播未经证实的传闻。

**重点词语**：
- 穿井：打井
- 溉汲：打水浇田。溉：浇灌、灌溉。汲：从井里取水
- 及：待，等到
- 道：讲述
- 闻之于宋君：使宋国的国君知道这件事。闻：知道，听说，这里是"使知道"的意思
- 对：应答，回答
- 得一人之使：得到一个人使唤，指得到一个人的劳力

**古今异义**：
- 穿：古义（挖掘）/ 今义（穿着）（穿井）
- 及：古义（等到）/ 今义（以及）（及其家穿井）

**写作特色**：
- 故事简明：短短数语，把事情讲清楚
- 寓意深刻：揭示以讹传讹的危害
- 启示性强：引导人们审慎对待传闻

2. **句子翻译**：全文翻译或重点句翻译
3. **内容理解**：概括故事内容
4. **寓意理解**：分析寓言的寓意
5. **古今异义**：辨析"穿""及"的古义今义

### 4.4 《杞人忧天》

#### 内容
**出处**：《列子·天瑞》

**原文**：
```
杞国有人忧天地崩坠，身亡所寄，废寝食者。

又有忧彼之所忧者，因往晓之，曰："天，积气耳，亡处亡气。若屈伸呼吸，终日在天中行止，奈何忧崩坠乎？"

其人曰："天果积气，日月星宿，不当坠耶？"

晓之者曰："日月星宿，亦积气中之有光耀者，只使坠，亦不能有所中伤。"

其人曰："奈地坏何？"

晓之者曰："地，积块耳，充塞四虚，亡处亡块。若躇步跐蹈，终日在地上行止，奈何忧其坏？"

其人舍然大喜，晓之者亦舍然大喜。
```

**翻译**：
杞国有个人担忧天会崩塌地会陷落，自己无处容身，便整天睡不好觉，吃不下饭。

又有一个人为这个杞国人的忧愁而忧愁，就去开导他，说："天不过是积聚的气体罢了，没有哪个地方没有空气的。你一举一动，一呼一吸，整天都在天空里活动，怎么还担心天会塌下来呢？"

那个人说："天如果是气体，那日月星辰不就会掉下来吗？"

开导他的人说："日月星辰也是空气中发光的东西，即使掉下来，也不会伤害什么。"

那个人又说："那地坏了又怎么办呢？"

开导他的人说："地不过是堆积的土块罢了，填满了四处，没有什么地方是没有土块的。你行走跳跃，整天都在地上活动，怎么还担心地会陷下去呢？"

那个杞国人放下心来，很高兴；开导他的人也放了心，很高兴。

**寓意**：讽刺那些为不必要或缺乏根据的事情而忧虑和担心的人。告诫人们：不要为没有根据的事情忧虑。

**重点词语**：
- 杞（qǐ）：杞国，周代诸侯国
- 崩坠：崩塌，坠落
- 身亡（wú）所寄：没有地方存身。亡：同"无"，没有
- 废寝食：睡不着觉，吃不下饭
- 因：于是，就
- 晓：告知，开导
- 积气：积聚的气体
- 若：你
- 行止：行动，活动
- 奈何：为何，为什么
- 果：果真
- 星宿（xiù）：泛指星辰
- 只使：纵使，即使
- 中（zhòng）伤：伤害
- 奈……何：把……怎么办
- 地坏：地陷
- 积块：堆积的土块
- 四虚：四方
- 躇（chú）步跐（cǐ）蹈：泛指人的站立行走。躇：立。步：走。跐：踩。蹈：跳
- 舍（shì）然：消除疑虑的样子。舍：同"释"，解除、消除

**通假字**：
- 亡：同"无"，没有（身亡所寄）
- 舍：同"释"，解除、消除（舍然大喜）

**写作特色**：
- 对话为主：通过对话展开情节、阐明道理
- 讽刺手法：讽刺杞人的无端忧虑
- 寓意深刻：告诫人们不要庸人自扰

**成语**：
- 杞人忧天：比喻不必要的或缺乏根据的忧虑和担心

#### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 单元：第六单元
- 页码：第129-130页

**85%**

#### 常见题型
1. **词语解释**：解释"崩坠""亡""晓""若""奈何""中伤""舍"等词语
2. **通假字**：指出"亡""舍"的通假
3. **句子翻译**：全文翻译或重点句翻译
4. **内容理解**：概括故事内容
5. **寓意理解**：分析寓言的寓意
6. **成语**：解释"杞人忧天"的含义

#### 注意事项

## 知识点5：单元写作——发挥联想和想象

### 内容
**写作要求**：
1. 理解联想和想象的区别
2. 学会运用联想和想象
3. 想象要合理，要有新意

**联想与想象的区别**：
- **联想**：由一事物想到与之相关的另一事物
  - 相似联想：由街灯想到明星（相似点：亮、多）
  - 相关联想：由中秋想到团圆（相关）
  - 对比联想：由炎热想到寒冷（相反）
  - 因果联想：由下雪想到寒冷（因果）
- **想象**：在原有形象的基础上创造出新形象
  - 再造想象：根据描述在头脑中形成形象
  - 创造想象：独立创造新形象

**联想和想象的方法**：
1. **夸张**：夸大事物的特征
2. **拟人**：赋予物以人的情感
3. **比喻**：抓住相似点
4. **梦幻**：利用梦境展开想象

**写作技巧**：
1. 想象要大胆，但也要合理
2. 要表达真情实感
3. 要有中心，不能乱想
4. 语言要生动形象

- 教材：义务教育教科书《语文》七年级上册（部编版）
- 单元：第六单元
- 页码：第136-137页

### 中考出题概率
**写作素养积累，间接考查**

### 注意事项
- 联想要自然，想象要合理
- 学习《天上的街市》的联想方法
- 童话写作要运用夸张和想象

## 知识点6：课外古诗词诵读

### 6.1 《秋词（其一）》刘禹锡

**原文**：
```
自古逢秋悲寂寥，我言秋日胜春朝。
晴空一鹤排云上，便引诗情到碧霄。
```

**翻译**：
自古以来，人们每逢秋天就悲叹秋天的寂寞萧索，我却说秋天远远胜过春天。秋日天高气爽，晴空万里，一只仙鹤推开云层直冲云霄，也激发我的诗情飞向万里晴空。

**创作背景**：
诗人被贬朗州时所作。古代文人多悲秋，诗人却赞秋，表现了乐观豁达的情怀。

**主题**：赞美秋天的明丽开阔，表现了诗人积极乐观、奋发向上的精神。

**艺术特色**：
- 对比手法：与传统悲秋对比
- 意象鲜明：晴空、仙鹤、碧霄
- 格调昂扬：一反悲秋传统

**重点词语**：
- 寂寥：寂寞空虚
- 春朝：春天
- 排云：推开白云
- 碧霄：蓝天

**中考出题概率**：75%

### 6.2 《夜雨寄北》李商隐

**原文**：
```
君问归期未有期，巴山夜雨涨秋池。
何当共剪西窗烛，却话巴山夜雨时。
```

**翻译**：
你问我回家的日期，我说归期实在难定。今晚巴山下着大雨，雨水已涨满秋池。什么时候我们才能一起在西窗下剪烛夜谈，再来叙说今天巴山夜雨的情景呢？

**创作背景**：
诗人客居巴蜀时写给远在长安的妻子（或友人）的诗。

**主题**：表达了游子对亲人（或友人）的思念之情。

**艺术特色**：
- 时空交错：此时的巴山夜雨 ↔ 未来的西窗夜话
- 虚实结合：实写眼前景，虚写未来情
- 含蓄蕴藉：不言思念而思念自现

**重点词语**：
- 寄北：寄给北方的人（妻子或友人）
- 归期：回家的日期
- 未有期：没有确定的日期
- 巴山：泛指巴蜀之地
- 何当：何时将要
- 剪烛：剪去燃焦的烛芯，使灯光明亮
- 却话：回头说，追述

**中考出题概率**：85%

### 6.3 《十一月四日风雨大作（其二）》陆游

**原文**：
```
僵卧孤村不自哀，尚思为国戍轮台。
夜阑卧听风吹雨，铁马冰河入梦来。
```

**翻译**：
我直挺挺躺在孤寂荒凉的乡村里，没有为自己的处境而感到悲哀，心中还想着替国家防卫边疆。夜深了，我躺在床上听到那风雨的声音，迷迷糊糊地梦见自己骑着披着铁甲的战马跨过冰封的河流出征北方疆场。

**创作背景**：
南宋光宗绍熙三年（1192年），诗人68岁，退居山阴老家。虽年迈体衰，仍心系国家。

**主题**：表达了诗人虽然年老体衰，但仍想为国戍边的爱国情怀。

**艺术特色**：
- 对比手法："僵卧孤村"与"铁马冰河"对比
- 虚实结合：实写风雨，虚写梦境
- 爱国情怀：至死不渝

**重点词语**：
- 僵卧：躺卧不起，形容老病
- 孤村：孤寂荒凉的村庄
- 戍：守卫
- 轮台：古代边疆地名，这里泛指边疆
- 夜阑：夜深
- 铁马：披着铁甲的战马
- 冰河：冰封的河流

**中考出题概率**：80%

### 6.4 《潼关》谭嗣同

**原文**：
```
终古高云簇此城，秋风吹散马蹄声。
河流大野犹嫌束，山入潼关不解平。
```

**翻译**：
自古以来高高云层就聚集在这座雄关之上，秋风阵阵总是吹散哒哒的马蹄声。奔腾而过的黄河与辽阔的原野还嫌太过约束，从华山进入潼关后更不知什么是平坦。

**创作背景**：
诗人18岁时途经潼关所作。潼关地势险要，山河雄伟。

**主题**：通过描写潼关雄伟壮丽的景色，表现了诗人冲破束缚、追求自由的少年意气。

**艺术特色**：
- 拟人手法："嫌束""不解平"赋予山河以人的情感
- 气势磅礴：高云、大河、高山
- 托物言志：借山河表达志向

**重点词语**：
- 终古：久远
- 簇：簇拥
- 马蹄声：骑马经过的声音
- 河流：指黄河
- 大野：辽阔的原野
- 嫌束：嫌弃束缚
- 不解平：不知道什么是平坦

**中考出题概率**：70%
