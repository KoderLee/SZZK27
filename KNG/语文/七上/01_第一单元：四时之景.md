# 第一单元：四时之景 - 知识点梳理

> 科目：语文 | 年级：七年级上册 | 教材：部编版/人教版
> 深圳中考出题概率：综合85%

---

## 知识点1：朱自清《春》

### 内容
**作者介绍**：朱自清（1898-1948），原名自华，字佩弦，江苏扬州人，现代著名散文家、诗人、学者。代表作有《背影》《荷塘月色》等。

**文章主旨**：本文是一篇写景抒情散文，描绘了春天到来时万物复苏、生机勃勃的景象，表达了作者对春天的热爱和赞美之情，以及对美好生活的向往。

**结构层次**：
- 盼春（第1段）：开篇点题，表达对春天的期盼
- 绘春（第2-7段）：细致描绘春天的各种景物
  - 春草图（第3段）：草的嫩、绿、多、软
  - 春花图（第4段）：花的繁、艳、甜
  - 春风图（第5段）：风的柔和、芳香、和悦
  - 春雨图（第6段）：雨的细密、轻盈、绵长
  - 迎春图（第7段）：人的活动，春到人欢
- 赞春（第8-10段）：用三个比喻赞美春天

**修辞手法**：
- 比喻："春天像刚落地的娃娃""像小姑娘""像健壮的青年"
- 拟人："小草偷偷地从土里钻出来""太阳的脸红起来了"
- 排比："红的像火，粉的像霞，白的像雪"

**重点词语**：
- 朗润：明朗润泽
- 酝酿：原指造酒的发酵过程，文中指各种气息在空气里像发酵似的，越来越浓
- 窠巢：鸟兽昆虫的窝
- 呼朋引伴：呼唤朋友，招引同伴
- 花枝招展：形容女子打扮得十分艳丽，文中比喻姿态优美

### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 单元：第一单元
- 页码：第2-5页（课文主体），第6页（课后习题）

### 中考出题概率
**90%**

### 常见题型
1. **默写题**：默写全文或重点段落（如"盼望着，盼望着……"开头段，结尾三段比喻句）
2. **词语理解**：解释"酝酿""窠巢""花枝招展"等词语在文中的含义
3. **修辞分析**：分析比喻、拟人等修辞手法的作用（如三个比喻句的妙处）
4. **内容理解**：概括春草、春花、春风、春雨的特点
5. **情感主旨**：分析作者表达的思想感情

### 注意事项
- 全文背诵要求，重点段落必须准确默写
- 注意"酝酿""窠巢""嘹亮"等易错字的写法
- 三个比喻句的顺序和内涵（新、美、力）是高频考点
- 景物描写角度（视觉、听觉、嗅觉、触觉）常结合考查

## 知识点2：老舍《济南的冬天》

### 内容
**作者介绍**：老舍（1899-1966），原名舒庆春，字舍予，北京人，满族，现代著名作家。代表作有小说《骆驼祥子》《四世同堂》，话剧《茶馆》《龙须沟》等。1951年被授予"人民艺术家"称号。

**文章主旨**：本文是一篇写景散文，描绘了济南冬天"温晴"的特点，表现了济南冬天的秀美景色，表达了作者对济南冬天的喜爱和赞美之情。

**结构层次**：
- 总写（第1段）：对比北平、伦敦、热带，突出济南冬天"温晴"的特点
- 分写（第2-5段）：
  - 阳光朗照下的山（第2段）：温暖舒适
  - 薄雪覆盖下的山（第3段）：秀美动人
  - 城外的远山（第4段）：淡雅如诗
  - 济南的水（第5段）：不结冰，冒着点热气，绿萍水藻
- 总结（第6段）：这就是冬天的济南

**写作特色**：
- 对比手法：与北平、伦敦、热带对比突出温晴
- 比喻拟人："这一圈小山在冬天特别可爱，好像是把济南放在一个小摇篮里"
- 情景交融：在景物描写中融入真挚情感

**重点词语**：
- 响晴：晴朗无云
- 温晴：温暖晴朗
- 安适：安静而舒适
- 秀气：小巧而清秀
- 贮蓄：储存，积聚
- 澄清：清澈明亮

### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 单元：第一单元
- 页码：第6-8页（课文主体），第9页（课后习题）

### 中考出题概率
**85%**

### 常见题型
1. **默写题**：默写第3段（薄雪覆盖下的山）或第5段（济南的水）
2. **内容理解**：济南冬天的特点是什么？作者是如何表现的？
3. **修辞分析**：分析"小摇篮""看护妇"等比喻的表达效果
4. **词语品析**：分析"卧""镶"等动词的妙处
5. **情感分析**：作者为什么说济南是"宝地"？

### 注意事项
- 重点段落（第3段、第5段）要求背诵默写
- 注意"髻""镶""贮蓄""澄清"等易错字
- "温晴"是贯穿全文的文眼
- 第3段空间顺序（山上→山尖→山坡→山腰）是常考点

## 知识点3：刘湛秋《雨的四季》

### 内容
**作者介绍**：刘湛秋（1935-2023），安徽芜湖人，当代著名诗人、翻译家、评论家。被誉为"抒情诗之王"。

**文章主旨**：本文描绘了春夏秋冬四季雨的不同特点，抒发了作者对雨的喜爱和赞美之情，表达了对生命的热爱。

**结构层次**：
- 总起（第1段）：我喜欢雨，雨给我的形象和记忆永远是美的
- 分写（第2-5段）：
  - 春雨：清新、甜美、生机（娇媚、透明、清新、甜美）
  - 夏雨：热烈、粗犷、奔放（热烈而又粗犷）
  - 秋雨：端庄、沉静、深情（端庄而又沉静）
  - 冬雨：自然、平静、纯洁（自然、平静、纯洁）
- 总结（第6-7段）：赞美雨，渴望生命活跃

**写作特色**：
- 抓住特征：准确抓住四季雨的不同特点
- 多种感官：视觉、听觉、嗅觉结合
- 诗化语言：语言优美，富有诗意

**重点词语**：
- 娇媚：撒娇讨人喜欢的样子
- 粗犷：粗豪、豪放
- 端庄：端正庄重
- 静谧：安静
- 高邈：高而远
- 莅临：来到，来临（多用于贵宾）

### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第9-11页

### 中考出题概率
**45%**

**说明**：本课为自读课文，深圳中考一般不直接考查，但其中的写景方法和词语理解可能在阅读理解中间接考查。

### 常见题型
1. **内容理解**：概括四季雨的特点
2. **词语理解**：解释"娇媚""粗犷""莅临"等词语
3. **语言品析**：赏析文中优美语句
4. **情感主旨**：分析作者对雨的情感

### 注意事项
- 不要求全文背诵，但需熟读理解
- 四季雨的特点要准确区分
- 诗化语言的特点需体会把握

## 知识点4：古代诗歌四首

### 4.1 《观沧海》曹操

#### 内容
**作者介绍**：曹操（155-220），字孟德，东汉末年政治家、军事家、诗人。他是建安文学的开创者，与曹丕、曹植合称"三曹"。

**创作背景**：建安十二年（207年），曹操北征乌桓得胜回师途中，登临碣石山时所作。

**诗歌内容**：
```
东临碣石，以观沧海。
水何澹澹，山岛竦峙。
树木丛生，百草丰茂。
秋风萧瑟，洪波涌起。
日月之行，若出其中；
星汉灿烂，若出其里。
幸甚至哉，歌以咏志。
```

**主题思想**：通过描写大海吞吐日月、包蕴万千的壮丽景象，表现了诗人开阔的胸襟和统一中原、建功立业的远大抱负。

**艺术特色**：
- 借景抒情：借大海的雄伟壮丽抒发政治抱负
- 虚实结合：实景与想象相结合
- 动静结合：海水"澹澹"与山岛"竦峙"，洪波"涌起"与草木"丰茂"

#### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第14页
- 出处：《步出夏门行·第一章》

#### 中考出题概率
**95%**

#### 常见题型
1. **默写题**：全诗默写，重点考查"日月之行……若出其里"
2. **炼字题**：赏析"观""涌"等字的妙处
3. **名句赏析**：分析"日月之行，若出其中；星汉灿烂，若出其里"的妙处
4. **情感分析**：诗人表达了怎样的思想感情？
5. **表现手法**：分析虚实结合、借景抒情的手法

#### 注意事项
- 必须全文熟练背诵默写
- 注意"碣石""澹澹""竦峙""萧瑟""星汉"等易错字
- 最后两句"幸甚至哉，歌以咏志"是乐府诗的套语，与正文内容无关
- "日月之行"四句是想象之景，体现诗人博大胸怀

### 4.2 《闻王昌龄左迁龙标遥有此寄》李白

#### 内容
**作者介绍**：李白（701-762），字太白，号青莲居士，又号"谪仙人"，唐代伟大的浪漫主义诗人，被后人誉为"诗仙"。

**创作背景**：王昌龄被贬为龙标尉，李白闻讯后作此诗寄赠，表达对友人的同情和牵挂。

**诗歌内容**：
```
杨花落尽子规啼，
闻道龙标过五溪。
我寄愁心与明月，
随君直到夜郎西。
```

**主题思想**：通过寄情明月，表达了对友人被贬远行的同情、牵挂和慰藉之情。

**艺术特色**：
- 寓情于景：杨花、子规营造凄凉氛围
- 想象奇特：将愁心寄与明月，随风追随友人
- 情景交融：景中含情，情由景生

**重点词语**：
- 左迁：贬官，降职
- 龙标：唐代县名，在今湖南黔阳
- 杨花：柳絮
- 子规：即杜鹃鸟，相传其啼声哀婉，似说"不如归去"
- 五溪：湘西五条溪流的总称
- 夜郎：唐代县名，在今湖南芷江

#### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第14页

#### 中考出题概率
**90%**

#### 常见题型
1. **默写题**：全诗默写，重点考查后两句
2. **意象分析**：分析"杨花""子规""明月"的意象作用
3. **名句赏析**：赏析"我寄愁心与明月，随君直到夜郎西"
4. **情感分析**：诗人表达了怎样的情感？
5. **炼字题**：赏析"尽""啼""寄"等字的表达效果

#### 注意事项
- 必须全文熟练背诵默写
- 注意"杨""道""溪""寄""郎"等易错字
- "左迁"是重要文化常识（右迁为升职，左迁为降职）
- 后两句是全诗精华，常考名句默写和理解

### 4.3 《次北固山下》王湾

#### 内容
**作者介绍**：王湾，生卒年不详，唐代诗人，洛阳人。现存诗10首，最著名的是《次北固山下》。

**创作背景**：诗人舟泊北固山下，看到潮平岸阔、残夜归雁的景象，触发思乡之情而作。

**诗歌内容**：
```
客路青山外，行舟绿水前。
潮平两岸阔，风正一帆悬。
海日生残夜，江春入旧年。
乡书何处达？归雁洛阳边。
```

**主题思想**：通过描写长江下游开阔秀丽的景色，抒发了旅途中的思乡之情。

**艺术特色**：
- 对仗工整：颔联、颈联对仗精妙
- 写景传神："潮平两岸阔，风正一帆悬"写景如画
- 蕴含哲理："海日生残夜，江春入旧年"富有哲理

**重点词语**：
- 次：停宿
- 客路：旅人前行的路
- 潮平：潮水涨满，与岸齐平
- 风正：顺风
- 残夜：夜将尽未尽之时
- 旧年：指年末、年终
- 乡书：家信

#### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第15页

#### 中考出题概率
**90%**

#### 常见题型
1. **默写题**：全诗默写，重点考查颔联、颈联
2. **名句赏析**：赏析"潮平两岸阔，风正一帆悬"或"海日生残夜，江春入旧年"
3. **炼字题**：赏析"生""入""阔""悬"等字
4. **情感分析**：诗人表达了怎样的情感？
5. **哲理理解**："海日生残夜，江春入旧年"蕴含了什么哲理？

#### 注意事项
- 必须全文熟练背诵默写
- 注意"潮""阔""悬""残""雁"等易错字
- 颈联"海日生残夜，江春入旧年"是千古名句，蕴含新旧交替、时光流逝的哲理
- "次"是停宿的意思，不是次数

### 4.4 《天净沙·秋思》马致远

#### 内容
**作者介绍**：马致远（约1251-1321以后），号东篱，元代著名戏曲作家、散曲家，与关汉卿、郑光祖、白朴并称"元曲四大家"。

**体裁介绍**：《天净沙》是曲牌名，属越调；"秋思"是题目。这是一首小令（散曲的一种）。

**作品内容**：
```
枯藤老树昏鸦，
小桥流水人家，
古道西风瘦马。
夕阳西下，
断肠人在天涯。
```

**主题思想**：通过描绘一幅萧瑟苍凉的深秋晚景图，表现了漂泊天涯的游子思念故乡、倦于漂泊的孤寂愁苦之情。

**艺术特色**：
- 意象叠加：前三句九个名词意象叠加，构成画面
- 以景托情：景物描写烘托游子的愁苦心情
- 对比手法："人家"的温馨与"断肠人"的凄凉形成对比

**重点词语**：
- 昏鸦：黄昏时归巢的乌鸦
- 古道：古老荒凉的道路
- 西风：秋风
- 瘦马：因长途跋涉而消瘦的马
- 断肠人：形容伤心悲痛到极点的人，此处指漂泊天涯的游子
- 天涯：天边，指远离家乡的地方

#### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第15页
- 出处：《全元散曲》

#### 中考出题概率
**95%**

#### 常见题型
1. **默写题**：全曲默写
2. **意象分析**：分析作品中出现的意象及其作用
3. **名句赏析**：赏析"夕阳西下，断肠人在天涯"
4. **情感分析**："断肠人"为什么"断肠"？表达了怎样的情感？
5. **写作手法**：分析意象叠加、以景托情的手法

#### 注意事项
- 必须全文熟练背诵默写
- 注意"藤""鸦""瘦""肠""涯"等易错字
- 前三句纯用名词意象，没有动词，这是独特之处
- "小桥流水人家"以乐景写哀情，反衬游子孤寂
- 被誉为"秋思之祖"

## 知识点5：单元写作——热爱生活，热爱写作

### 内容
**写作要求**：
1. 学会观察生活，发现生活中的美
2. 善于捕捉触动心灵的瞬间
3. 运用多种感官进行描写
4. 写出真情实感

**写作技巧**：
1. **观察角度**：视觉、听觉、嗅觉、味觉、触觉
2. **观察方法**：定点观察、移步换景、动静结合
3. **写景顺序**：空间顺序（由远及近、由上到下）、时间顺序
4. **表达方式**：记叙、描写、抒情相结合

### 出处
- 教材：义务教育教科书《语文》七年级上册（部编版）
- 页码：第17-18页

### 中考出题概率
**写作素养积累，间接考查**

### 常见题型
- 本单元为写作指导，不单独命题，但写作技巧在作文中考查

### 注意事项
- 写景要抓住景物特征
- 要融入真情实感
- 学习课文中的写景方法（比喻、拟人等）

## 知识点10：修辞手法专项归纳

### 本单元常见修辞手法

| 修辞 | 定义 | 例句 | 作用 |
|:-----|:-----|:-----|:-----|
| 比喻 | 用相似的事物打比方 | 春天像刚落地的娃娃 | 生动形象，化抽象为具体 |
| 拟人 | 把物当作人来写 | 小草偷偷地从土里钻出来 | 赋予物以人的情态，生动亲切 |
| 排比 | 结构相同或相似的句子排列 | 红的像火，粉的像霞，白的像雪 | 增强语势，条理清晰 |
| 对偶 | 字数相等、结构对称 | 两个黄鹂鸣翠柳，一行白鹭上青天（类此） | 句式整齐，音韵和谐 |
| 夸张 | 对事物进行扩大或缩小 | 日月之行，若出其中 | 突出特征，增强感染力 |
| 反复 | 重复某一词语或句子 | 盼望着，盼望着 | 强调情感，增强节奏感 |

### 《春》修辞赏析要点

**比喻三层次（结尾段）**
- 春天像刚落地的娃娃——从头到脚都是新的，他生长着（**新**）
- 春天像小姑娘——花枝招展的，笑着，走着（**美**）
- 春天像健壮的青年——有铁一般的胳膊和腰脚，他领着我们上前去（**力**）

**答题模板**：
运用了比喻的修辞手法，把春天比作娃娃/小姑娘/青年，生动形象地写出了春天新/美/力的特点，表达了作者对春天的喜爱和赞美之情。

### 《济南的冬天》修辞赏析要点

**拟人例句**："这一圈小山在冬天特别可爱，好像是把济南放在一个小摇篮里，它们全安静不动地低声地说：'你们放心吧，这儿准保暖和。'"

**答题模板**：
运用了拟人的修辞手法，赋予小山以人的动作和语言，生动形象地写出了济南被小山环绕的地理特点，表现了济南冬天温晴的特点，表达了作者对济南冬天的喜爱之情。

## 知识点12：课后习题要点

### 《春》课后习题要点

- 参考答案：春草图、春花图、春风图、春雨图、迎春图

**思考探究二**：课文读起来富有童趣，又带有诗的味道，清新，活泼，优美。你有没有这样的感觉？试找出一些段落细加品味，并跟同学分享你的体会。
- 答题要点：从修辞（比喻、拟人）、语言（叠词、短句）、情感（对春天的热爱）角度分析

**思考探究三**：作者把春天比作"刚落地的娃娃""小姑娘""健壮的青年"，你怎样理解这些比喻？你还能发挥想象，另写一些比喻句来描绘春天吗？
- 答题要点：三个比喻分别体现春天的新、美、力；表现了春天的成长过程

**积累拓展四**：结合具体语境，分析下面句子中加点词语的表达效果。
- "盼望着，盼望着"——反复，表达急切期盼之情
- "小草偷偷地从土里钻出来"——拟人，"偷偷地""钻"生动写出小草悄然而生的情态

### 《济南的冬天》课后习题要点

**思考探究一**：作者所写的济南的冬天有着怎样的特点？他是通过哪些景物呈现这一特点的？
- 参考答案：温晴；通过阳光朗照下的山、薄雪覆盖下的山、城外的远山、济南的水呈现

## 知识点13：易错点专项提醒

### 默写易错点

| 易错句 | 常见错误 | 正确写法 | 记忆技巧 |
|:-------|:---------|:---------|:---------|
| 盼望着，盼望着 | 望忘混淆 | 盼望 | 用"目"望 |
| 东风来了，春天的脚步近了 | 东冬混淆 | 东风 | 春风指东风 |
| 红的像火，粉的像霞，白的像雪 | 顺序颠倒 | 红粉白 | 与桃花、杏花、梨花对应 |
| 春天像刚落地的娃娃 | 的得混淆 | 的 | 名词前用"的" |
| 从头到脚都是新的 | 从丛混淆 | 从头 | 从（介词） |
| 水何澹澹 | 淡淡混淆 | 澹澹 | 水波荡漾貌，从水 |
| 山岛竦峙 | 悚峙混淆 | 竦峙 | 竦：高耸 |
| 我寄愁心与明月 | 与于混淆 | 与（给） | 给明月 |
| 随君直到夜郎西 | 郎朗混淆 | 夜郎 | 古地名 |
| 潮平两岸阔 | 阔扩混淆 | 开阔 | 宽阔 |
| 风正一帆悬 | 悬玄混淆 | 悬（挂） | 帆挂着 |
| 断肠人在天涯 | 崖涯混淆 | 天涯 | 天边，从水 |

### 理解易错点

1. **《春》结尾三段顺序不能颠倒**
   - 娃娃（新）→ 小姑娘（美）→ 青年（力）
   - 这是按照人的成长过程排列，体现春天从诞生到成熟的过程

2. **《观沧海》最后两句不是正文**
   - "幸甚至哉，歌以咏志"是乐府诗的套语，与正文内容无关
   - 答题时不能将其作为写景内容分析

3. **《天净沙·秋思》前三句无动词**
   - "枯藤老树昏鸦，小桥流水人家，古道西风瘦马"
   - 这是意象叠加的手法，纯用名词构成画面

4. **左迁是贬官**
   - 汉代以右为尊，左迁即贬官
   - 王昌龄被贬为龙标尉，所以李白写诗寄赠

5. **"次"是停宿的意思**
   - 《次北固山下》即停宿在北固山下
   - 不是"一次""两次"的意思

### 答题易错点

| 错误答法 | 正确答法 | 说明 |
|:---------|:---------|:-----|
| 用了比喻的修辞 | 运用了比喻的修辞手法，把……比作……，生动形象地写出了……，表达了…… | 要有具体分析 |
| 表达了作者的感情 | 表达了作者对春天的喜爱和赞美之情 | 情感要具体 |
| 词语写得很好 | "钻"字运用了拟人的修辞，生动形象地写出了小草破土而出的情态 | 要分析词语的表达效果 |
