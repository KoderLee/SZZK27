## 知识点1：茨威格《伟大的悲剧》

### 内容
**作者介绍**：斯蒂芬·茨威格（1881-1942），奥地利著名作家。代表作有《象棋的故事》《一个陌生女人的来信》《昨日的世界》等。

**写作背景**：本文是作者为1911年英国探险家斯科特率领的南极探险队而作。斯科特一行虽到达南极点，但晚于挪威探险家阿蒙森，归途中不幸遇难。

**文章主旨**：通过记叙斯科特探险队悲壮覆灭的故事，赞扬了人类勇于探索、为事业献身的崇高精神和强烈的集体主义精神。

**结构层次**：
- 第一部分（第1-3段）：启程奔向南极点
- 第二部分（第4-7段）：到达南极点，发现阿蒙森已先到
- 第三部分（第8-10段）：返程遇难
  - 埃文斯精神失常而死去
  - 奥茨不愿拖累同伴而独自走向暴风雪
  - 斯科特等三人冻死在帐篷里
- 第四部分（第11-16段）：英雄之死及后世纪念

**重点语句赏析**：
1. "一个人虽然在同不可战胜的厄运的搏斗中毁灭了自己，但他的心灵却因此变得无比高尚。"
   - 评价：点明悲剧的伟大意义
   - 毁灭的是肉体，升华的是精神

2. "挪威国旗耀武扬威、洋洋得意地在这被人类冲破的堡垒上猎猎作响。"
   - 拟人：反衬斯科特一行的沮丧
   - "耀武扬威""洋洋得意"：表达复杂心情

3. "他希望以后会有人在他的遗体旁发现这些能证明他和英国民族勇气的日记。"
   - 细节：表现斯科特的尊严和责任感

**人物形象**：
- 斯科特：
  - 坚毅执着：勇于探险
  - 诚实守信：为阿蒙森作证
  - 集体主义：关心同伴
  - 绅士风度：坦然面对失败

**写作特色**：
- 细节描写：日记、书信等
- 对比手法：阿蒙森的成功与斯科特的失败
- 直抒胸臆：作者直接评价
- 悲剧色彩：悲壮感人

**重点词语**：
- 拽：拉
- 无垠：没有边际
- 癫狂：精神错乱，发疯
- 凛冽：极为寒冷，严寒刺骨
- 吞噬：吞食
- 销蚀：消损腐蚀
- 羸弱：瘦弱
- 步履：脚步
- 告罄：指物资用完
- 遗孀：某人死后，他的妻子称为某人的遗孀

## 知识点2：杨利伟《太空一日》

### 内容
**作者介绍**：杨利伟（1965-），辽宁绥中人。中国第一位进入太空的航天员。2003年10月15日乘坐"神舟五号"飞船进入太空。

**写作背景**：本文选自杨利伟自传《天地九重》，回忆了"神舟五号"飞天过程中的经历和感受。

**文章主旨**：通过真实记录太空一日的经历，展现了中国航天员勇敢无畏、严谨科学的精神风貌，激发了民族自豪感。

**结构层次**：
- 第一部分（第1-4段）：起飞前的准备
- 第二部分（第5-17段）：升空过程
  - 火箭发射
  - 经历"共振"痛苦
  - 进入太空
- 第三部分（第18-25段）：太空飞行
  - 失重体验
  - 观察地球
  - 神秘的敲击声
- 第四部分（第26-39段）：返回过程
  - 飞船返航
  - 穿越黑障
  - 成功着陆

**重点语句赏析**：
1. "我曾俯瞰我们的首都北京，白天它是燕山山脉边的一片灰白，分辨不清，夜晚则呈现一片红晕。"
   - 细节：真实的太空观察
   - 体现航天员对祖国的深情

2. "那种共振持续26秒后，慢慢减轻。"
   - 科学数据：体现纪实性
   - "26秒"：虽然短暂却感觉漫长

3. "就这一下，指挥大厅有人大声喊道：'快看啊，他眨眼了，利伟还活着！'"
   - 侧面描写：表现地面人员的关心
   - 生动真实

**写作特色**：
- 真实记录：第一手太空经历
- 科学严谨：数据准确
- 语言朴实：不加雕饰
- 情感真挚：爱国情怀

**重点词语**：
- 炽热：温度高，极热
- 轮廓：构成图形或物体的外缘的线条
- 俯瞰：俯视，从高处往下看
- 模拟：模仿
- 严谨：严密谨慎
- 稠密：多而密
- 烧灼：烧、烫，使受伤
- 五脏六腑：人体内脏器官的统称
- 千钧重负：比喻很沉重的负担或非常重大的责任

## 知识点3：刘慈欣《带上她的眼睛》

### 内容
**作者介绍**：刘慈欣（1963-），山西阳泉人。中国科幻小说代表作家。代表作有《三体》《流浪地球》等。

**写作背景**：本文是刘慈欣的科幻短篇小说，创作于1999年。

**文章主旨**：通过"我"带着一位被困地心的女宇航员的"眼睛"游览地面的故事，歌颂了人类探索未知世界的勇敢精神，以及面对绝境时的乐观和坚守。

**结构层次**：
- 第一部分（第1-5段）：带上她的"眼睛"
  - "我"去旅行，带上传感眼镜
  - 与小姑娘共享感官
- 第二部分（第6-16段）：游览中的感受
  - 小姑娘对一切充满好奇和热爱
  - "我"被她的情感感染
- 第三部分（第17-23段）：了解真相
  - 得知她是"落日六号"的领航员
  - 被困地心，无法返回
- 第四部分（第24-26段）：永远的思念
  - 小姑娘继续在地下工作
  - "我"永远怀念她

**重点语句赏析**：
1. "我感到，她对这个世界的情感已丰富到病态的程度。"
   - 伏笔：为后文揭示真相做铺垫
   - "病态"：实则是极度热爱

2. "周围是温度高达5000摄氏度，压力可以把碳在一秒钟内变成金刚石的液态铁镍！"
   - 科学数据：体现科幻小说的科学性
   - 渲染绝境的恐怖

3. "无论走到天涯海角，我离她都不会再远了。"
   - 结尾：含蓄深沉
   - "我"会永远怀念她

**写作特色**：
- 悬念设置：逐步揭示真相
- 科幻元素：传感眼镜、地心航行等
- 人文关怀：对人的情感和命运的关注
- 诗意表达：科幻与抒情结合

**重点词语**：
- 迟钝：（感官、思想、行动等）反应慢，不灵敏
- 闲暇：闲空
- 凸现：清楚地显现
- 拍摄：用摄影机或摄像机把人、物的形象记录下来
- 蔚蓝：像晴朗的天空的颜色
- 吟唱：吟咏歌唱
- 点缀：加以衬托或装饰，使原有事物更加美好
- 漫步：没有目的而悠闲地走
- 孤零零：形容孤单，无依无靠或没有陪衬

## 知识点4：文言文——《活板》

### 内容
**作者介绍**：沈括（1031-1095），字存中，浙江钱塘人。北宋著名科学家、政治家。著有《梦溪笔谈》。

**作品信息**：《梦溪笔谈》是一部涉及古代中国自然科学、工艺技术及社会历史现象的综合性笔记体著作。

**文章主旨**：通过介绍活字印刷术的制作和使用方法，赞扬了我国古代劳动人民的智慧和创造力。

**原文与翻译**：（略，详见教材）

**重点词语**：
- 板印：指雕版印刷
- 盛：大规模地
- 五经：指《诗》《书》《礼》《易》《春秋》五部儒家经典
- 典籍：记载古代法制的图书，泛指古代图书
- 毕昇：北宋发明家，活字印刷术发明人
- 活板：活字版
- 其上以松脂、蜡和纸灰之类冒之：在它上面用松脂、蜡混合着纸灰这一类东西覆盖好
- 冒：覆盖
- 持就火炀之：把它拿到火上烘烤
- 药：指上文说的松脂、蜡等物
- 字平如砥：字印平整得像磨刀石
- 止：同"只"
- 未为简易：不能算是简便
- 数十百千本：几十、成百、上千本
- 自：另自，另外
- 奇字：生僻字
- 素：平素，向来
- 瞬息可就：很短的时间就能完成
- 具：准备好
- 更互：交替，轮流
- 帖：同"贴"
- 韵：指韵部
- 木格：木框格子
- 奇字：写法特殊，或生僻、不常用的字
- 用讫：用完
- 火令药熔：用火烤使药物熔化
- 殊不：一点也不

**通假字**：
- 板：同"版"
- 已：同"以"
- 止：同"只"
- 帖：同"贴"

**写作特色**：
- 说明顺序清晰：制字→制版→印刷→拆版
- 语言准确：运用列数字、作比较等说明方法
- 详略得当：详写制版过程，略写其他

**活字印刷术的步骤**：
1. 制字：用胶泥刻字，火烧使坚
2. 制版：板上置铁范，密布字印
3. 固定：用松脂、蜡和纸灰覆盖，火烤使熔
4. 印刷：一版印刷，一版布字，交替使用
5. 拆版：火烤使药熔，用手拂去

## 知识点5：单元写作——语言简明

### 内容
**写作要求**：
1. 避免重复啰嗦
2. 避免表意不明
3. 合理安排详略

**语言简明的方法**：
1. **删除冗余**：删去不必要的词语
2. **避免重复**：不重复表达同一意思
3. **长句改短**：复杂句子简单化
4. **准确表意**：用词准确，不模棱两可

**示例**：
- 原句：我亲眼看见他亲自把书放到书架上。
- 修改：我看见他把书放到书架上。

## 知识点6：名著导读——《海底两万里》

### 内容
**作品信息**：
- 作者：儒勒·凡尔纳（1828-1905），法国著名科幻小说家，被誉为"科幻小说之父"
- 体裁：科幻小说
- 创作时间：1869-1870年
- 主要内容：讲述鹦鹉螺号潜艇的海底探险故事

**主要人物**：
1. **尼摩船长**：
   - 身份：鹦鹉螺号潜艇的船长
   - 性格：神秘、智慧、勇敢、反抗压迫
   - 特点：知识渊博，厌恶陆地文明

2. **阿龙纳斯**：
   - 身份：法国生物学家
   - 性格：博学、好奇、善良
   - 角色：故事的叙述者

3. **康塞尔**：
   - 身份：阿龙纳斯的仆人
   - 性格：忠诚、沉稳、分类学专家

4. **尼德·兰**：
   - 身份：加拿大捕鲸手
   - 性格：野性、急躁、渴望自由

**主要情节**：
- 阿龙纳斯受邀参加追捕"海怪"
- 发现"海怪"是潜水艇，被尼摩船长囚禁
- 跟随鹦鹉螺号进行海底旅行
- 见识海底奇观，经历种种险情
- 最终逃脱，回到陆地

**主题思想**：
1. 赞美科学探索精神
2. 反对殖民压迫
3. 呼吁保护海洋环境
4. 展现人类对未知世界的好奇

**艺术特色**：
1. **科学性与幻想性结合**：基于科学知识的合理想象
2. **场面描写壮观**：海底世界的瑰丽描绘
3. **悬念设置巧妙**：尼摩船长的神秘身份
4. **人物形象鲜明**：各具特色的人物

**经典场景**：
- 海底森林狩猎
- 珊瑚墓地
- 南极冰下被困
- 大战章鱼

## 知识点7：课外古诗词诵读

### 7.1 《泊秦淮》杜牧

**原文**：
```
烟笼寒水月笼沙，夜泊秦淮近酒家。
商女不知亡国恨，隔江犹唱后庭花。
```

**主题**：借陈后主荒淫亡国的历史，讽刺晚唐统治者的醉生梦死，表达对国家命运的忧虑。

**重点词语**：
- 秦淮：秦淮河，流经南京
- 商女：歌女
- 后庭花：乐曲名，陈后主所作，被视为亡国之音

**中考出题概率**：80%

### 7.2 《贾生》李商隐

**原文**：
```
宣室求贤访逐臣，贾生才调更无伦。
可怜夜半虚前席，不问苍生问鬼神。
```

**主题**：借贾谊的遭遇，讽刺统治者求贤而不能真正用贤，表达对怀才不遇的感慨。

**重点词语**：
- 贾生：贾谊，西汉政治家、文学家
- 宣室：汉代宫殿名
- 逐臣：被贬谪的大臣
- 苍生：百姓

**中考出题概率**：75%

### 7.3 《过松源晨炊漆公店（其五）》杨万里

**原文**：
```
莫言下岭便无难，赚得行人错喜欢。
政入万山围子里，一山放出一山拦。
```

**主题**：通过写山区行路的感受，说明人生道路上困难重重的道理，告诫人们不要为眼前的顺境所迷惑。

**重点词语**：
- 松源、漆公店：地名
- 赚得：骗得
- 错喜欢：空欢喜
- 政：同"正"
- 放出：放过

**中考出题概率**：70%

### 7.4 《约客》赵师秀

**原文**：
```
黄梅时节家家雨，青草池塘处处蛙。
有约不来过夜半，闲敲棋子落灯花。
```

**主题**：写诗人雨夜等候友人不至的情景，表现了闲适恬淡的心境和些许怅惘之情。

**重点词语**：
- 黄梅时节：夏初江南梅子黄熟的时节
- 约客：约请客人
- 过夜半：过了半夜
- 闲敲棋子：无聊地敲着棋子

**中考出题概率**：75%
