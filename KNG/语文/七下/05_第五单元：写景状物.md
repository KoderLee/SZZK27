## 知识点1：宗璞《紫藤萝瀑布》

### 内容
**作者介绍**：宗璞（1928-），原名冯钟璞，河南唐河人。当代著名女作家。代表作有《红豆》《三生石》《丁香结》等。

**写作背景**：1982年，作者弟弟身患绝症，作者悲痛万分，徘徊于庭院中，看见一树盛开的紫藤萝花，悟到人生的美好和生命的永恒，于是写下此文。

**文章主旨**：通过描写紫藤萝花的盛开，感悟到生命的美好和永恒，表达了作者对生命的热爱和珍惜，以及积极面对人生的态度。

**结构层次**：
- 第一部分（第1-6段）：赏花
  - 描写紫藤萝花的繁盛美丽
  - 从整体到局部，从花色到花香
- 第二部分（第7-9段）：忆花
  - 回忆十多年前紫藤萝花的凋零
  - 对比现在花的繁盛
- 第三部分（第10-11段）：悟花
  - 感悟生命的永恒
  - 表达积极的人生态度

**重点语句赏析**：
1. "从未见过开得这样盛的藤萝，只见一片辉煌的淡紫色，像一条瀑布，从空中垂下，不见其发端，也不见其终极。"
   - 比喻：把紫藤萝比作瀑布
   - 夸张："不见其发端，也不见其终极"
   - 表现花的繁盛壮观

2. "紫色的大条幅上，泛着点点银光，就像迸溅的水花。仔细看时，才知道那是每一朵紫花中的最浅淡的部分，在和阳光互相挑逗。"
   - 拟人："互相挑逗"
   - 生动形象地写出花的活泼可爱

3. "花和人都会遇到各种各样的不幸，但是生命的长河是无止境的。"
   - 主旨句：点明文章中心
   - 哲理：生命永恒，要积极面对人生

**写作特色**：
- 托物言志：借紫藤萝花表达对生命的感悟
- 比喻拟人：生动形象地描写紫藤萝
- 情景交融：在景物描写中融入情感
- 对比手法：现在花的繁盛与过去花的凋零对比

**重点词语**：
- 瀑布：从山壁上或河床突然降落的地方流下的水，远看好像挂着的白布
- 迸溅：向四外溅
- 挑逗：逗引，招惹
- 忍俊不禁：忍不住笑
- 仙露琼浆：比喻美酒或甘美的浆汁
- 蜂围蝶阵：蜜蜂包围，蝴蝶排列成阵
- 盘虬卧龙：形容枝干弯弯曲曲，苍劲有力

### 常见题型
1. **内容理解**：概括紫藤萝花的特点
2. **写作手法**：分析托物言志的作用
3. **语句赏析**：赏析"像一条瀑布"等比喻句
4. **主旨理解**：分析"生命的长河是无止境的"的含义
5. **情感变化**：分析作者情感的变化过程

- 掌握比喻、拟人等修辞手法

## 知识点2：贾平凹《一棵小桃树》

### 内容
**作者介绍**：贾平凹（1952-），陕西丹凤人。当代著名作家。代表作有《秦腔》《古炉》《废都》等。

**写作背景**：本文写于1976年，作者借一棵小桃树的成长经历，表达了自己在人生道路上追求理想、顽强奋斗的历程。

**文章主旨**：通过描写一棵小桃树在逆境中顽强生长的经历，表现了作者对理想的执着追求和不屈不挠的奋斗精神。

**结构层次**：
- 第一部分（第1段）：眼前的小桃树（引子）
- 第二部分（第2-10段）：回忆小桃树的来历和成长
  - 奶奶买来桃子，留下桃核
  - 种下桃核，发芽生长
  - 被遗忘，在角落里艰难生长
- 第三部分（第11-13段）：现在的小桃树
  - 在风雨中挣扎
  - 顶端还保留着一个花苞
- 第四部分（第14段）：由树及人，表达志向

**重点语句赏析**：
1. "它长得很委屈，是弯了头，紧抱了身子的。"
   - 拟人：赋予小桃树以人的情态
   - "委屈"：表现小桃树生长的艰难

2. "雨还在下着，我的小桃树千百次地俯下身去，又千百次地挣扎起来。"
   - 反复："千百次"强调次数之多
   - 表现小桃树顽强的生命力

3. "我心里稍稍有了些安慰。啊，小桃树啊！我该怎么感激你？"
   - 抒情：直接表达对小桃树的感激
   - 由树及人，感悟人生

**写作特色**：
- 托物言志：借小桃树表达人生追求
- 双线并行：明线写树，暗线写人
- 对比手法：小桃树的柔弱与顽强对比
- 拟人手法：赋予小桃树以人的情感

**重点词语**：
- 褪：脱（衣服、羽毛、颜色等）
- 忏悔：认识了过去的错误或罪过而感觉痛心
- 哆嗦：因受外界刺激而身体不由自主地颤动
- 矜持：拘谨，拘束
- 执着：原为佛教用语，指对某一事物坚持不放，不能超脱。后来指固执或拘泥，也指坚持不懈
- 服侍：伺候，照料
- 渺小：微小
- 魂魄：迷信的人指附在人体内可以脱离人体存在的精神
- 幼稚：年纪小；形容头脑简单或缺乏经验
- 颤抖：哆嗦，发抖

## 知识点3：古代诗歌五首

### 3.1 《登幽州台歌》陈子昂

#### 内容
**作者介绍**：陈子昂（661-702），字伯玉，梓州射洪（今四川射洪）人。唐代著名诗人，初唐诗文革新的代表人物。

**写作背景**：武则天万岁通天元年（696年），陈子昂随军出征，因献计不被采纳，反遭贬斥，登幽州台（今北京附近），感怀而作。

**诗歌内容**：
```
前不见古人，后不见来者。
念天地之悠悠，独怆然而涕下。
```

**翻译**：
往前不见古代招贤的圣君，向后不见后世求才的明君。想到天地的广阔无边，我独自悲伤，潸然泪下。

**主题**：表达了诗人怀才不遇、孤独寂寞的苦闷心情，以及对理想破灭的悲愤。

**艺术特色**：
- 意境雄浑：天地悠悠，孤独怆然
- 直抒胸臆：直接表达内心情感
- 语言质朴：不加雕饰，感人至深

**重点词语**：
- 幽州台：即黄金台，又称蓟北楼，故址在今北京市大兴
- 古人：指古代那些能够礼贤下士的圣君
- 来者：指后世那些重视人才的贤明君主
- 悠悠：形容时间的久远和空间的广大
- 怆然：悲伤的样子
- 涕：眼泪

### 3.2 《望岳》杜甫

#### 内容
**作者介绍**：杜甫（712-770），字子美，河南巩县人。唐代伟大的现实主义诗人，被后世尊为"诗圣"。

**写作背景**：唐玄宗开元二十三年（735年），杜甫到洛阳应进士考试，落第后开始漫游。此诗为漫游齐赵时所作。

**诗歌内容**：
```
岱宗夫如何？齐鲁青未了。
造化钟神秀，阴阳割昏晓。
荡胸生曾云，决眦入归鸟。
会当凌绝顶，一览众山小。
```

**翻译**：
泰山到底怎么样呢？它苍翠的山色在齐鲁大地连绵不断。大自然把神奇和秀美都集中于泰山，山南山北，明暗不同。层层云气升腾，荡涤胸襟；张目远望，归鸟入山。我一定要登上泰山的顶峰，俯瞰那众山，而众山就会显得极为渺小。

**主题**：表达了诗人对泰山的赞美之情，以及不怕困难、敢于攀登绝顶、俯视一切的雄心和气概。

**艺术特色**：
- 写景层次分明：远望→近望→凝望→俯望
- 气势磅礴："一览众山小"
- 寓情于景：在写景中表达雄心壮志

**重点词语**：
- 岱宗：泰山。岱：泰山的别称。宗：五岳之首
- 夫如何：怎么样
- 齐鲁：春秋时的两个诸侯国，在今山东一带
- 青未了：青翠的山色连绵不断
- 造化：指天地、大自然
- 钟：聚集
- 神秀：神奇和秀美
- 阴阳：山北水南为阴，山南水北为阳
- 割：分割
- 昏晓：黄昏和早晨
- 荡胸：心胸震荡
- 曾：同"层"
- 决眦：极力张大眼睛。眦：眼眶
- 会当：一定要
- 凌：登上
- 绝顶：最高峰
- 一览众山小：化用《孟子·尽心上》"登泰山而小天下"

### 3.3 《登飞来峰》王安石

#### 内容
**作者介绍**：王安石（1021-1086），字介甫，号半山，抚州临川（今江西抚州）人。北宋著名政治家、文学家、思想家。

**写作背景**：宋仁宗皇祐二年（1050年），王安石任鄞县知县期满，回江西临川故里，途经杭州，登飞来峰而作。

**诗歌内容**：
```
飞来山上千寻塔，闻说鸡鸣见日升。
不畏浮云遮望眼，自缘身在最高层。
```

**翻译**：
飞来峰上有座很高的塔，听说鸡鸣时分可以看见旭日升起。不怕浮云遮住我远望的视线，只因为我已经站在最高的层顶。

**主题**：表达了诗人高瞻远瞩、不畏艰险的进取精神，以及变法革新的政治抱负。

**艺术特色**：
- 写景与议论结合
- 比喻巧妙："浮云"比喻奸佞小人或困难障碍
- 哲理深刻：站得高才能看得远

**重点词语**：
- 飞来峰：浙江绍兴城外的宝林山，传说此山是从天外飞来
- 千寻塔：很高的塔。寻：古代长度单位，八尺为一寻
- 闻说：听说
- 鸡鸣见日升：鸡鸣时分可看到日出
- 畏：怕
- 浮云：飘浮的云彩，比喻奸佞小人或眼前的困难障碍
- 遮望眼：挡住远望的视线
- 缘：因为
- 最高层：最高处

### 3.4 《游山西村》陆游

#### 内容
**作者介绍**：陆游（1125-1210），字务观，号放翁，越州山阴（今浙江绍兴）人。南宋著名爱国诗人。

**写作背景**：宋孝宗乾道三年（1167年），陆游被罢官闲居在家乡山阴，游山西村时所作。

**诗歌内容**：
```
莫笑农家腊酒浑，丰年留客足鸡豚。
山重水复疑无路，柳暗花明又一村。
箫鼓追随春社近，衣冠简朴古风存。
从今若许闲乘月，拄杖无时夜叩门。
```

**翻译**：
不要笑农家腊月里酿的酒浑浊，丰收年景款待客人，菜肴丰足。山峦重叠水流曲折，正担心无路可走，柳绿花艳忽然眼前又出现一个山村。吹着箫打起鼓，春社的日子已经接近，村民们衣冠简朴，古代风气仍然保存。今后如果还能趁着月明来闲游，我一定拄着拐杖随时来敲你的家门。

**主题**：描写了山西村优美的自然风光和淳朴的民风，表达了诗人对田园生活的喜爱和向往。

**艺术特色**：
- 写景与叙事结合
- 对仗工整："山重水复"对"柳暗花明"
- 名句千古："山重水复疑无路，柳暗花明又一村"
- 蕴含哲理：困境中往往蕴含希望

**重点词语**：
- 腊酒：腊月里酿造的酒
- 浑：浑浊
- 足鸡豚：准备了丰盛的菜肴。足：充足，丰盛。豚：小猪，这里指猪肉
- 山重水复：山峦重叠，水流曲折
- 柳暗花明：柳色深绿，花色红艳
- 箫鼓：吹箫打鼓
- 春社：古代立春后祭祀土地神的日子
- 古风：古代风俗
- 闲乘月：趁着月明来闲游
- 拄杖：拄着拐杖
- 无时：随时
- 叩门：敲门

### 3.5 《己亥杂诗（其五）》龚自珍

#### 内容
**作者介绍**：龚自珍（1792-1841），字璱人，号定庵，浙江仁和（今杭州）人。清代著名思想家、文学家。

**写作背景**：清道光十九年（1839年），作者辞官南归，后又北上迎接眷属，往返途中写成《己亥杂诗》315首，这是第五首。

**诗歌内容**：
```
浩荡离愁白日斜，吟鞭东指即天涯。
落红不是无情物，化作春泥更护花。
```

**翻译**：
浩浩荡荡的离别愁绪向着日落西斜的远处延伸，马鞭向东一挥，感觉就是人在天涯一般。从枝头上掉下来的落花不是无情之物，即使化作春泥，也甘愿培育美丽的春花成长。

**主题**：表达了诗人辞官离京时的复杂心情，以及虽然辞官但仍关心国家命运、愿为国效力的奉献精神。

**艺术特色**：
- 抒情与议论结合
- 比喻巧妙："落红"自喻，"春泥护花"表达奉献精神
- 意境开阔：离愁浩荡，天涯遥远

**重点词语**：
- 浩荡：广阔无边的样子，这里形容离愁
- 离愁：离别时的愁绪
- 白日斜：太阳西斜
- 吟鞭：诗人的马鞭
- 东指：向东指（作者从北京回杭州）
- 天涯：形容离京遥远
- 落红：落花
- 更：更加
- 护花：培育花朵

## 知识点4：单元写作——文从字顺

### 内容
**写作要求**：
1. 语句表达要准确
2. 语句表达要连贯
3. 语句表达要得体

**文从字顺的方法**：
1. **用词准确**：
   - 辨析词义
   - 注意搭配
   - 避免生造词语

2. **语句通顺**：
   - 成分完整
   - 语序合理
   - 关联恰当

3. **语意连贯**：
   - 话题统一
   - 顺序合理
   - 过渡自然

4. **修改润色**：
   - 朗读检查
   - 推敲词句
   - 请他人审阅
