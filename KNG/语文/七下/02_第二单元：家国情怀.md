## 知识点1：光未然《黄河颂》

### 内容
**作者介绍**：光未然（1913-2002），原名张光年，湖北光化人。现代著名诗人、文学评论家。代表作有《黄河大合唱》等。

**写作背景**：1939年，光未然率领抗敌演剧队第三队从延安奔赴晋察冀边区，途经黄河时，被黄河的雄伟气势所震撼，创作了《黄河大合唱》歌词。后经冼星海谱曲，成为抗日救亡运动的经典之作。

**文章主旨**：通过歌颂黄河的雄伟气势和历史贡献，激发中华儿女的民族自豪感和爱国热情，号召人们学习黄河的伟大坚强，保卫祖国。

**结构层次**：
- 朗诵词（序曲）：点明歌颂黄河的主题
- 歌词（主体）：
  - 望黄河（我站在高山之巅……劈成南北两面）
    - 近景：惊涛澎湃，掀起万丈狂澜
    - 全景：浊流宛转，结成九曲连环
    - 远景：从昆仑山下/奔向黄海之边
    - 横向：把中原大地/劈成南北两面
  - 颂黄河（啊！黄河！……伸出千万条铁的臂膀）
    - 养育中华民族（摇篮）
    - 保卫中华民族（屏障）
    - 激励中华民族（精神）
  - 学黄河（我们祖国的英雄儿女……像你一样的伟大坚强！）
    - 学习黄河精神
    - 保卫祖国

**写作特色**：
- 直抒胸臆：直接抒发对黄河的赞美
- 比喻象征：
  - 摇篮——养育
  - 屏障——保卫
  - 臂膀——力量
- 反复咏叹："啊！黄河！"三次出现
- 第二人称：面对面倾诉，情感强烈

**重点语句赏析**：
1. "望黄河滚滚，奔向东南。惊涛澎湃，掀起万丈狂澜；浊流宛转，结成九曲连环。"
   - 动词连用：望、奔、掀、结
   - 描绘黄河磅礴气势

2. "啊！黄河！你是中华民族的摇篮！"
   - 比喻：黄河比作摇篮
   - 象征：黄河是中华民族的发源地

3. "啊！黄河！你是伟大坚强，像一个巨人出现在亚洲平原之上，用你那英雄的体魄，筑成我们民族的屏障。"
   - 比喻：黄河比作巨人、屏障
   - 象征：黄河是民族精神的象征

4. "我们祖国的英雄儿女，将要学习你的榜样，像你一样的伟大坚强！像你一样的伟大坚强！"
   - 反复：强调学习黄河精神
   - 点明主旨

**重点词语**：
- 巅：山顶
- 澎湃：形容波浪互相撞击
- 狂澜：巨大的波浪，比喻动荡不定的局势或猛烈的潮流
- 屏障：像屏风那样遮挡着的东西（多指山岭、岛屿等）
- 哺育：喂养，比喻培养
- 九曲连环：形容黄河弯道多
- 浩浩荡荡：形容水势浩大汹涌
- 气魄：气势
- 坚强：强固有力，不可动摇或摧毁

- 教材：义务教育教科书《语文》七年级下册（部编版）
- 单元：第二单元

### 常见题型
1. **默写题**：默写"我站在高山之巅……"或"啊！黄河！你是中华民族的摇篮"等段落
2. **写作手法**：分析比喻、象征的作用
3. **词语理解**：解释"狂澜""屏障""哺育"等词语
4. **内容理解**：概括黄河的象征意义
5. **情感主旨**：分析诗歌表达的爱国情感

- 掌握反复咏叹的修辞手法

## 知识点2：陆定一《老山界》

### 内容
**作者介绍**：陆定一（1906-1996），江苏无锡人，无产阶级革命家。长征途中任红军总政治部宣传部部长。

**写作背景**：本文是作者回忆1934年红军长征途中翻越老山界的经历。

**文章主旨**：通过记叙红军翻越老山界的经过，表现了红军不怕困难、乐观向上的革命精神和坚强的革命意志。

**结构层次**：
- 第一部分（第1段）：介绍老山界，决定翻山
- 第二部分（第2-32段）：翻山经过
  - 下午动身（第2-11段）：从山脚往山上走，天色已晚
  - 夜里行军（第12-22段）：在山腰露宿，半夜冻醒
  - 黎明下山（第23-30段）：继续爬山，到达山顶
  - 下午下山（第31-32段）：顺利下山
- 第三部分（第33段）：总结感受

**重点语句赏析**：
1. "满天都是星光，火把也亮起来了。从山脚向上望，只见火把排成许多'之'字形，一直连到天上，跟星光接起来，分不出是火把还是星星。"
   - 环境描写：星光与火把交织
   - "之"字形：表现山路的曲折陡峭
   - "连到天上"：夸张手法，表现山势高峻

2. "耳朵里有不可捉摸的声响，极远的又是极近的，极洪大的又是极细切的，像春蚕在咀嚼桑叶，像野马在平原上奔驰，像山泉在呜咽，像波涛在澎湃。"
   - 排比、比喻：四个比喻写出深山中各种声响
   - 以声衬静：用声音衬托夜晚的寂静
   - 表现红军战士的乐观精神

3. "路上有几处景致很好，浓密的树林里，银子似的泉水流下山去，清得透底。"
   - 景物描写：表现红军欣赏美景的闲适心情
   - 体现革命乐观主义精神

4. "老山界是我们长征中所过的第一座难走的山。但是我们走过了。"
   - 总结：点明老山界的"难"
   - "但是我们走过了"：表现红军的自豪和自信

**写作特色**：
- 按时间顺序：下午→夜里→黎明→下午
- 景物描写：景物与情感交融
- 以小见大：通过一件小事表现红军精神
- 语言朴实：真实感人

**重点词语**：
- 攀谈：闲谈
- 峭壁：像墙壁一样陡的山崖
- 骨碌：滚动
- 咀嚼：用牙齿磨碎食物
- 呜咽：低声哭泣
- 督促：监督催促
- 灌输：输送（思想、知识等）
- 苛捐杂税：指繁重的捐税
- 酣然入梦：畅快地入睡
- 不可捉摸：指对人或事物无法猜测和估量
- 满望：十分希望

### 出处
- 教材：义务教育教科书《语文》七年级下册（部编版）
- 单元：第二单元
- 页码：第31-36页

### 中考出题概率
**80%**

### 常见题型
1. **内容理解**：概括翻越老山界的经过
2. **写作手法**：分析景物描写的作用
3. **语句赏析**：赏析"像春蚕在咀嚼桑叶……"等句子
4. **人物形象**：分析红军战士的形象特点
5. **情感主旨**：分析文章表现的革命精神

### 注意事项
- 理解按时间顺序记叙的方法
- 赏析景物描写的句子
- 注意"攀""峭""嚼""呜""酣"等易错字

- 体会红军的革命乐观主义精神

## 知识点3：端木蕻良《土地的誓言》

### 内容
**作者介绍**：端木蕻良（1912-1996），辽宁昌图人，现代著名作家。代表作有《科尔沁旗草原》《大地的海》等。

**写作背景**：1941年9月18日，"九一八"事变十周年之际，作者怀着对故乡的深深眷恋和对侵略者的无比愤恨，写下了这篇散文。

**文章主旨**：通过抒发对故乡土地的热爱和思念，表达了作者誓死保卫家乡、收复失地的决心和强烈的爱国主义情感。

**结构层次**：
- 第一部分（第1段）：抒发对土地的誓言
- 第二部分（第2-5段）：回忆故乡的美丽丰饶
  - 第2段：故乡的景色
  - 第3段：故乡的生活
  - 第4段：故乡的物产
  - 第5段：故乡的召唤
- 第三部分（第6-7段）：表达收复失地的决心

**写作特色**：
- 直抒胸臆：直接抒发对土地的热爱
- 排比反复：增强语势，情感强烈
- 景物描写：描绘故乡美景，反衬失地之痛
- 呼告手法：直接与土地对话

**重点语句赏析**：
1. "对于广大的关东原野，我心里怀着挚痛的热爱。"
   - 开篇点题，直接抒情
   - "挚痛"：真挚而沉痛

2. "我无时无刻不听见她呼唤我的名字，无时无刻不听见她召唤我回去。"
   - 双重否定：强调无时无刻都在思念
   - 拟人：土地在"呼唤""召唤"

3. "我必须看见一个更美丽的故乡出现在我的面前——或者我的坟前。"
   - 誓言：誓死收复失地
   - 悲壮：不惜牺牲生命

**重点词语**：
- 炽痛：热烈而深切
- 泛滥：江河湖泊的水溢出，比喻坏的事物不受限制地流行
- 斑斓：灿烂多彩
- 怪诞：离奇古怪
- 亘古：远古
- 默契：双方的意思没有明白说出而彼此有一致的了解
- 田垄：田埂
- 蚱蜢：昆虫，像蝗虫
- 污秽：肮脏的东西
- 耻辱：声誉上所受的损害

### 出处
- 教材：义务教育教科书《语文》七年级下册（部编版）
- 单元：第二单元
- 页码：第37-39页

### 中考出题概率
**75%**

### 常见题型
1. **内容理解**：概括作者对故乡土地的情感
2. **写作手法**：分析排比、呼告的作用
3. **语句赏析**：赏析"我必须看见……"等句子
4. **情感主旨**：分析文章的爱国情感
5. **背景理解**：结合"九一八"事变背景理解文章

### 注意事项
- 理解文章写作背景（"九一八"事变十周年）
- 体会作者对故乡的深厚感情
- 注意"炽""斑斓""诞""亘""默契""污秽"等易错字
- 理解双重否定句的表达效果

## 知识点4：《木兰诗》

### 内容
**作品介绍**：《木兰诗》又称《木兰辞》，是南北朝时北方的一首乐府民歌，与《孔雀东南飞》并称为"乐府双璧"。

**文章主旨**：通过叙述木兰女扮男装、代父从军的故事，塑造了一个勇敢善良、勤劳孝顺、不慕荣利的巾帼英雄形象，反映了古代人民对和平生活的向往。

**原文与翻译**：

**第一段**：
```
唧唧复唧唧，木兰当户织。不闻机杼声，唯闻女叹息。
问女何所思，问女何所忆。女亦无所思，女亦无所忆。
昨夜见军帖，可汗大点兵，军书十二卷，卷卷有爷名。
阿爷无大儿，木兰无长兄，愿为市鞍马，从此替爷征。
```
- 翻译：叹息声一声接着一声，木兰对着门织布。听不到织布机的声音，只听到木兰的叹息声。问木兰在想什么，问木兰在思念什么。木兰没有想什么，木兰没有思念什么。昨夜看见军中的文告，可汗大规模征兵，征兵的名册有很多卷，每卷上都有父亲的名字。父亲没有大儿子，木兰没有兄长，愿意为此去买鞍马，从此替父亲出征。

**第二段**：
```
东市买骏马，西市买鞍鞯，南市买辔头，北市买长鞭。
旦辞爷娘去，暮宿黄河边，不闻爷娘唤女声，但闻黄河流水鸣溅溅。
旦辞黄河去，暮至黑山头，不闻爷娘唤女声，但闻燕山胡骑鸣啾啾。
```
- 翻译：到东市买骏马，到西市买马鞍和坐垫，到南市买马嚼子和缰绳，到北市买长鞭。早晨辞别父母离开，晚上宿营在黄河边，听不到父母呼唤女儿的声音，只听到黄河水奔腾流淌的声音。早晨辞别黄河离开，晚上到达黑山头，听不到父母呼唤女儿的声音，只听到燕山胡人的战马啾啾的鸣叫声。

**第三段**：
```
万里赴戎机，关山度若飞。朔气传金柝，寒光照铁衣。
将军百战死，壮士十年归。
```
- 翻译：不远万里奔赴战场，像飞一样跨过一道道关，越过一座座山。北方的寒气传送着打更的声音，清冷的月光照着战士们的铠甲。将士们身经百战，有的战死沙场，有的多年后胜利归来。

**第四段**：
```
归来见天子，天子坐明堂。策勋十二转，赏赐百千强。
可汗问所欲，木兰不用尚书郎，愿驰千里足，送儿还故乡。
```
- 翻译：回来朝见天子，天子坐在朝堂上。给木兰记了很大的功勋，赏赐了很多财物。可汗问木兰想要什么，木兰不愿做尚书郎，希望骑上千里马，送我回故乡。

**第五段**：
```
爷娘闻女来，出郭相扶将；阿姊闻妹来，当户理红妆；
小弟闻姊来，磨刀霍霍向猪羊。开我东阁门，坐我西阁床，
脱我战时袍，著我旧时裳。当窗理云鬓，对镜帖花黄。
出门看火伴，火伴皆惊忙：同行十二年，不知木兰是女郎。
```
- 翻译：父母听说女儿回来了，互相搀扶着出城迎接；姐姐听说妹妹回来了，对着门梳妆打扮；弟弟听说姐姐回来了，霍霍地磨刀杀猪宰羊。打开我东阁的门，坐在我西阁的床上，脱去我打仗时穿的战袍，穿上我以前的衣裳。对着窗户梳理美丽的头发，对着镜子贴上花黄。出门去看同伍的士兵，同伍的士兵都很惊讶：同行这么多年，竟然不知道木兰是女子。

**第六段**：
```
雄兔脚扑朔，雌兔眼迷离；双兔傍地走，安能辨我是雄雌？
```
- 翻译：雄兔两只脚时常动弹，雌兔两只眼睛时常眯着；雄雌两兔贴近地面跑，怎能分辨出哪只是雄兔，哪只是雌兔呢？

**人物形象**：
- 孝顺父母：替父从军
- 勇敢坚强：万里赴戎机，关山度若飞
- 不慕荣利：不用尚书郎，愿驰千里足
- 机智谨慎：同行十二年，不知木兰是女郎

**写作特色**：
- 详略得当：详写出征前、归乡后，略写战争过程
- 排比反复：增强语势，节奏感强
- 互文见义：上下句互相补充
- 比喻巧妙：以兔为喻，幽默含蓄

**重点词语**：
- 唧唧：叹息声
- 当户：对着门
- 机杼：织布机
- 军帖：军中的文告
- 可汗：古代西北地区少数民族对君主的称呼
- 点兵：征兵
- 十二：表示多数，不是确指
- 爷：父亲
- 市：买
- 鞍马：泛指马和马具
- 鞯：马鞍下的垫子
- 辔头：驾驭牲口用的嚼子和缰绳
- 旦：早晨
- 溅溅：水流声
- 啾啾：马叫声
- 戎机：战事
- 度：越过
- 朔气：北方的寒气
- 金柝：古时军中白天用来烧饭、夜里用来打更的器具
- 铁衣：铠甲
- 明堂：古代帝王举行大典的朝堂
- 策勋：记功
- 转：勋位每升一级叫一转
- 强：有余
- 不用：不愿做
- 尚书郎：尚书省的官员
- 郭：外城
- 红妆：指女子的艳丽装束
- 霍霍：磨刀的声音
- 著：穿
- 云鬓：像云那样的鬓发，形容好看的头发
- 帖花黄：贴上花黄。花黄：古代妇女的一种面部装饰物
- 火伴：同伍的士兵
- 扑朔：动弹
- 迷离：眯着眼
- 傍地走：贴近地面跑
- 安：怎么

**通假字**：
- 帖：同"贴"，贴上（对镜帖花黄）
- 火：同"伙"，同伴（出门看火伴）

**古今异义**：
- 爷：古义（父亲）/ 今义（爷爷）（卷卷有爷名）
- 走：古义（跑）/ 今义（行走）（双兔傍地走）
- 但：古义（只）/ 今义（但是）（但闻黄河流水鸣溅溅）

**一词多义**：
- 愿：
  - 愿意（愿为市鞍马）
  - 希望（愿驰千里足）

**成语**：
- 磨刀霍霍：形容准备动手干某事
- 扑朔迷离：形容事物错综复杂，难以辨别

**修辞手法**：
- 排比：东市买骏马……北市买长鞭
- 对偶：朔气传金柝，寒光照铁衣
- 互文：将军百战死，壮士十年归
- 比喻：双兔傍地走，安能辨我是雄雌

## 知识点5：单元写作——学习抒情

### 内容
**写作要求**：
1. 了解直接抒情和间接抒情
2. 学会运用恰当的抒情方式
3. 在写作中表达真情实感

**抒情方式**：
1. **直接抒情**：直接表达内心的情感
   - 特点：直白、强烈
   - 示例：《黄河颂》"啊！黄河！你是中华民族的摇篮！"

2. **间接抒情**：通过叙述、描写、议论等方式表达情感
   - 借景抒情：通过景物描写表达情感
   - 借事抒情：通过叙述事件表达情感
   - 借物抒情：通过描写事物表达情感
   - 示例：《老山界》通过景物描写表现红军的乐观精神

**抒情注意事项**：
1. 情感要真挚
2. 方式要恰当
3. 要为中心服务
4. 语言要生动
