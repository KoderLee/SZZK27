## 知识点1：鲁迅《阿长与〈山海经〉》

### 内容
**作者介绍**：鲁迅（1881-1936），原名周树人，字豫才，浙江绍兴人。中国现代伟大的文学家、思想家、革命家。

**作品出处**：《朝花夕拾》。

**写作背景**：本文是鲁迅回忆童年保姆阿长的散文，写于1926年。

**文章主旨**：通过回忆阿长的种种琐事，特别是她为"我"买来《山海经》的事，表现了阿长淳朴、善良、乐于助人的品格，表达了作者对这位普通劳动妇女的感激和怀念之情。

**结构层次**：
- 第一部分（第1-2段）：介绍阿长的身份和名字由来
- 第二部分（第3-18段）：记叙阿长的种种"毛病"
  - 喜欢切切察察
  - 睡相不好，摆成"大"字
  - 规矩繁琐
  - 讲"长毛"的故事
- 第三部分（第19-29段）：阿长为"我"买《山海经》
- 第四部分（第30-31段）：表达对阿长的怀念和感激

**重点语句赏析**：
1. "一到夏天，睡觉时她又伸开两脚两手，在床中间摆成一个'大'字。"
   - 细节描写：生动刻画阿长睡相的粗俗
   - 语言幽默，带有调侃意味

2. "我似乎遇着了一个霹雳，全体都震悚起来；赶紧去接过来，打开纸包，是四本小小的书，略略一翻，人面的兽，九头的蛇，……果然都在内。"
   - 心理描写："霹雳""震悚"表现"我"的震惊和激动
   - 细节描写：打开纸包、略略一翻
   - 体现《山海经》对"我"的重要性

3. "这四本书，乃是我最初得到，最为心爱的宝书。"
   - 评价：强调这四本书的意义
   - 原因：不仅是书本身，更凝聚着阿长的爱心

4. "仁厚黑暗的地母呵，愿在你怀里永安她的魂灵！"
   - 抒情：直接抒发对阿长的怀念和祝福
   - 情感达到高潮

**人物形象**：
- 阿长：
  - 淳朴善良：为"我"买《山海经》
  - 乐于助人：关心"我"的需求
  - 有点粗俗：睡相不好，喜欢切切察察
  - 有点迷信：讲究规矩，讲长毛故事
  - 地位卑微：连真实姓名都没有

**写作特色**：
- 欲扬先抑：先写阿长的"毛病"，后写她的善良
- 细节描写：通过具体细节表现人物
- 儿童视角：以童年眼光看待阿长
- 感情真挚：怀念之情溢于言表

**重点词语**：
- 谋死：谋杀
- 切切察察：细碎的说话声
- 絮说：絮絮叨叨地说
- 规矩：一定的标准、法则或习惯
- 郑重：严肃认真
- 惶急：恐惧着急
- 诘问：追问，责问
- 疏懒：懒散而不习惯于受拘束
- 震悚：身体因恐惧或过度兴奋而颤动
- 粗拙：粗糙低劣

3. **语句赏析**：赏析"我似乎遇着了一个霹雳……"等句子

## 知识点2：杨绛《老王》

### 内容
**作者介绍**：杨绛（1911-2016），原名杨季康，江苏无锡人。现代著名作家、翻译家。代表作有散文《干校六记》《我们仨》，译作《堂吉诃德》等。丈夫钱钟书。

**写作背景**：本文写于1984年，回忆作者一家与三轮车夫老王的交往。

**文章主旨**：通过回忆与老王交往的几个片段，表现了老王淳朴善良、知恩图报的品格，表达了作者对老王的同情、感激和愧疚之情，体现了人道主义精神。

**结构层次**：
- 第一部分（第1-4段）：介绍老王的基本情况
  - 职业：三轮车夫
  - 身体状况：有眼疾，瞎了一只眼
  - 家庭情况：无依无靠
  - 住所：破落大院
- 第二部分（第5-16段）：回忆与老王的交往
  - 老王为"我"家送冰
  - 老王送默存去医院
  - 老王送香油和鸡蛋
- 第三部分（第17-22段）：老王的去世和"我"的愧疚

**重点语句赏析**：
1. "他也许是从小营养不良而瞎了一眼，也许是得了恶病，反正同是不幸，而后者该是更深的不幸。"
   - 推测语气："也许"
   - 体现作者对老王的同情

2. "他送的冰比他前任送的大一倍，冰价相等。"
   - 对比：突出老王的憨厚老实
   - "前任"：杨绛式的幽默

3. "我强笑说：'老王，这么新鲜的大鸡蛋，都给我们吃？'"
   - 神态描写："强笑"——内心难过却不得不笑
   - 表现作者的复杂心情

4. "那是一个幸运的人对一个不幸者的愧怍。"
   - 点题：揭示文章主旨
   - 情感：对老王的愧疚和自责

**人物形象**：
- 老王：
  - 淳朴善良：送冰量大，送钱先生去医院不要钱
  - 知恩图报：临死前送香油和鸡蛋
  - 孤苦伶仃：无亲无故，身体残疾
  - 有尊严：不愿白受人好处

**写作特色**：
- 以小见大：通过小事表现人性光辉
- 平淡质朴：语言平实，情感深沉
- 对比手法：老王的淳朴与社会的冷漠对比
- 情感真挚：愧疚之情感人至深

**重点词语**：
- 惶恐：惊慌害怕
- 取缔：明令取消或禁止
- 降格：降低标准、身份等
- 镶嵌：把一物体嵌入另一物体内
- 滞笨：呆滞笨拙
- 侮辱：使对方人格或名誉受到损害，蒙受耻辱
- 愧怍：惭愧

### 出处
- 教材：义务教育教科书《语文》七年级下册（部编版）
- 单元：第三单元
- 页码：第58-61页

### 中考出题概率
**85%**

### 常见题型
1. **内容理解**：概括老王的性格特点
2. **写作手法**：分析以小见大的作用
3. **语句赏析**：赏析"那是一个幸运的人对一个不幸者的愧怍"
4. **情感主旨**：分析作者的情感变化
5. **人物形象**：分析老王的形象

### 注意事项
- 理解"幸运的人"与"不幸者"的含义
- 理解作者"愧怍"的原因
- 注意"取缔""镶嵌""滞""愧怍"等易错字
- 体会平淡语言中的深沉情感

## 知识点3：李森祥《台阶》

### 内容
**作者介绍**：李森祥（1956-），浙江衢州人，当代作家。代表作有《小学老师》《抒情年代》《情世诗文》等。

**写作背景**：本文是作者回忆父亲为建造高台阶新屋而奋斗的故事。

**文章主旨**：通过记叙父亲为建造有高台阶的新屋而辛苦劳作的一生，表现了农民勤劳、坚韧、要强的品格，以及他们对地位的渴望和尊重的需求。

**结构层次**：
- 第一部分（第1-8段）：父亲想要造有高台阶的新屋
  - 家乡台阶的文化含义
  - 父亲觉得自家台阶低
- 第二部分（第9-16段）：父亲为造屋做准备
  - 捡砖捡瓦
  - 存角票
  - 砍柴卖钱
- 第三部分（第17-24段）：造屋过程
  - 父亲的辛劳
  - 新屋建成
- 第四部分（第25-32段）：新屋建成后
  - 父亲老了
  - 台阶高了，但父亲不习惯

**重点语句赏析**：
1. "父亲总觉得我们家的台阶低。"
   - "总"字：表现父亲长期的渴望
   - 台阶的象征意义：社会地位

2. "父亲的两手没处放似的，抄着不是，贴在胯骨上也不是。"
   - 细节描写：表现父亲的不自在
   - 说明父亲不适应高台阶带来的"地位"

3. "怎么了呢，父亲老了。"
   - 结尾：感叹句式
   - 点明父亲为造屋付出的巨大代价

**人物形象**：
- 父亲：
  - 勤劳坚韧：多年辛苦劳作
  - 要强有志气：不甘于台阶低
  - 淳朴老实：典型的中国农民
  - 有尊严：希望通过高台阶获得尊重

**写作特色**：
- 细节描写：大量生活细节
- 象征手法：台阶象征社会地位
- 情感含蓄：在平实叙述中蕴含深情
- 以小见大：通过台阶反映农民心理

**重点词语**：
- 凼：水坑
- 门槛：门框下部挨着地面的横木（也有用石头的）
- 涎水：口水
- 筹划：想办法；定计划
- 晌午：中午
- 烦躁：烦闷急躁
- 头颅：人的头
- 尴尬：神色、态度不自然
- 微不足道：非常渺小，不值得一提
- 大庭广众：人很多的公开场合

- 单元：第三单元
- 页码：第62-68页

### 中考出题概率
**75%**

### 常见题型
1. **内容理解**：概括父亲的形象特点
2. **写作手法**：分析台阶的象征意义
3. **语句赏析**：赏析关键句的含义
4. **主旨理解**：分析文章主题
5. **细节分析**：分析父亲的不自在

### 注意事项
- 理解台阶的象征意义

## 知识点4：欧阳修《卖油翁》

### 内容
**作者介绍**：欧阳修（1007-1072），字永叔，号醉翁，晚号六一居士，吉州永丰（今江西永丰）人。北宋著名文学家、史学家。"唐宋八大家"之一。

**文章主旨**：通过陈尧咨射箭和卖油翁酌油的故事，说明"熟能生巧"的道理，告诫人们即使有专长，也不应骄傲自满。

**原文**：
```
陈康肃公尧咨善射，当世无双，公亦以此自矜。尝射于家圃，有卖油翁释担而立，睨之久而不去。见其发矢十中八九，但微颔之。

康肃问曰："汝亦知射乎？吾射不亦精乎？"翁曰："无他，但手熟尔。"康肃忿然曰："尔安敢轻吾射！"翁曰："以我酌油知之。"乃取一葫芦置于地，以钱覆其口，徐以杓酌油沥之，自钱孔入，而钱不湿。因曰："我亦无他，惟手熟尔。"康肃笑而遣之。
```

**翻译**：
康肃公陈尧咨擅长射箭，当时没有第二个人能和他相比，他也因此而自夸。曾经有一次，他在自家的园圃里射箭，有个卖油的老翁放下担子，站在一旁，斜着眼看他射箭，很久不离开。老翁见到他射出的箭十支能中八九支，只是微微地点点头。

陈尧咨问道："你也懂得射箭吗？难道我射箭的技艺不精湛吗？"老翁说："没有什么别的奥妙，只不过是手法熟练罢了。"陈尧咨气愤地说："你怎么能够轻视我射箭的本领！"老翁说："凭我倒油的经验知道这个道理。"于是老翁拿出一个葫芦放在地上，用一枚铜钱盖住葫芦的口，慢慢地用勺子倒油（通过铜钱方孔）注到葫芦里，油从铜钱的孔中注进去，却没有沾湿铜钱。接着老翁说："我也没有什么其他奥妙，只不过是手法熟练罢了。"康肃公笑着把老翁打发走了。

**人物形象**：
- 陈尧咨：
  - 射箭技艺高超：当世无双
  - 骄傲自满：以此自矜
  - 知错能改：笑而遣之
- 卖油翁：
  - 技艺高超：酌油不湿钱
  - 谦逊低调：无他，但手熟尔
  - 经验丰富：以我酌油知之

**重点词语**：
- 陈康肃公：陈尧咨，谥号康肃，北宋人
- 善射：擅长射箭
- 当世无双：当时没有第二个人
- 自矜：自夸
- 家圃：家里的园圃
- 释担：放下担子
- 睨：斜着眼看，形容不在意的样子
- 去：离开
- 发矢：射箭
- 但微颔之：只是微微点头。颔：点头
- 汝：你
- 无他：没有别的（奥妙）
- 但手熟尔：只是手法熟练罢了。尔：同"耳"，相当于"罢了"
- 忿然：气愤的样子
- 尔安敢轻吾射：你怎么敢轻视我射箭的本领
- 以我酌油知之：凭我倒油的经验知道这个（道理）
- 乃：于是，就
- 置：放
- 覆：盖
- 徐：慢慢地
- 杓：同"勺"
- 沥之：注入葫芦。沥：下滴
- 惟：只，不过
- 遣之：让他走。遣：打发

**通假字**：
- 尔：同"耳"，相当于"罢了"（但手熟尔）
- 杓：同"勺"（徐以杓酌油沥之）

**古今异义**：
- 但：古义（只，只是）/ 今义（但是）（但手熟尔）
- 安：古义（怎么）/ 今义（安全）（尔安敢轻吾射）
- 轻：古义（轻视）/ 今义（重量轻）（尔安敢轻吾射）

**一词多义**：
- 之：
  - 代词，指陈尧咨射箭（睨之久而不去）
  - 代词，指射箭十中八九的情况（但微颔之）
  - 代词，指卖油翁（康肃笑而遣之）
- 以：
  - 凭、靠（以我酌油知之）
  - 用（以钱覆其口）
  - 认为（公亦以此自矜）
- 而：
  - 表顺承（释担而立）
  - 表修饰（笑而遣之）

**成语**：
- 熟能生巧：熟练了就能产生巧办法，或找出窍门

**写作特色**：
- 对比手法：陈尧咨与卖油翁对比
- 语言简洁：全文仅133字
- 寓理于事：通过故事阐明道理
- 详略得当：详写酌油，略写射箭

### 常见题型
1. **默写/翻译**：全文翻译或重点句翻译
2. **词语解释**：解释"自矜""睨""但""颔""忿然""安""轻""徐""酌""遣"等词语
3. **通假字**：指出"尔""杓"的通假
4. **古今异义**：辨析"但""安""轻"的古义今义

## 知识点5：单元写作——抓住细节

### 内容
**写作要求**：
1. 学会观察细节
2. 学会描写细节
3. 让细节为文章中心服务

**细节描写的方法**：
1. **动作细节**：准确使用动词
   - 示例：《阿长与〈山海经〉》"伸开两脚两手，在床中间摆成一个'大'字"

2. **神态细节**：描写面部表情
   - 示例：《老王》"他面色死灰，两只眼上都结着一层翳"

3. **语言细节**：记录个性化语言
   - 示例：《卖油翁》"无他，但手熟尔"

4. **心理细节**：描写内心活动
   - 示例：《老王》"我强笑说"

**细节描写的作用**：
1. 刻画人物性格
2. 表现人物情感
3. 推动情节发展
4. 深化文章主题

## 知识点6：名著导读——《骆驼祥子》

### 内容
**作品信息**：
- 作者：老舍（1899-1966），原名舒庆春，字舍予，北京人，现代著名作家。
- 体裁：长篇小说
- 创作时间：1936-1937年
- 主要内容：描写人力车夫祥子的悲惨命运

**主要人物**：
1. **祥子**：
   - 身份：人力车夫
   - 性格变化：
     - 前期：勤劳、朴实、要强、有上进心
     - 后期：麻木、潦倒、狡猾、自暴自弃
   - 悲剧原因：社会压迫、个人奋斗的失败

2. **虎妞**：
   - 身份：车厂主刘四爷的女儿，祥子的妻子
   - 性格：泼辣、精明、自私、控制欲强

3. **小福子**：
   - 身份：祥子喜欢的女人
   - 命运：被生活所迫沦为妓女，最终自杀
   - 意义：祥子最后的希望破灭

4. **刘四爷**：
   - 身份：人和车厂老板，虎妞的父亲
   - 性格：霸道、自私、冷酷

**主要情节**：
1. **一起**：祥子来到北平当人力车夫，苦干三年，凑足一百块钱，买了辆新车
2. **一落**：连人带车被宪兵抓去当壮丁，理想第一次破灭
3. **二起**：卖骆驼，拼命拉车，省吃俭用攒钱准备买新车
4. **二落**：干包月时，辛苦攒的钱被孙侦探搜去，第二次希望破灭
5. **三起**：虎妞以低价给祥子买了邻居二强子的车
6. **三落**：为了置办虎妞的丧事，祥子又卖掉了车

**主题思想**：
1. 揭露旧社会对劳动人民的压迫和剥削
2. 批判个人奋斗道路在旧社会的行不通
3. 表达对劳动人民的同情

**艺术特色**：
1. **京味语言**：使用北京方言，生动传神
2. **人物塑造**：成功塑造祥子、虎妞等典型人物
3. **环境描写**：真实再现老北京的风貌
4. **心理描写**：细腻描写祥子的心理变化

**经典名句**：
- "雨下给富人，也下给穷人；下给义人，也下给不义的人。其实雨并不公道，因为下落在一个没有公道的世界上。"
- "经验是生活的肥料，有什么样的经验便变成什么样的人，在沙漠里养不出牡丹来。"

## 知识点7：课外古诗词诵读

### 7.1 《竹里馆》王维

**原文**：
```
独坐幽篁里，弹琴复长啸。
深林人不知，明月来相照。
```

**翻译**：
独自坐在幽深的竹林里，一边弹琴一边长声呼啸。深深的山林中无人知晓，只有一轮明月静静相照。

**主题**：表现了诗人宁静淡泊的心境和超然物外的情趣。

**艺术特色**：
- 用字简朴："独""幽""深""明"
- 意境清幽：竹林、深林、明月
- 动静结合：弹琴长啸 vs 幽篁深林

**重点词语**：
- 幽篁：幽深的竹林
- 长啸：撮口发出清越悠长的声音
- 相照：相对而视，互相映照

**中考出题概率**：70%

### 7.2 《春夜洛城闻笛》李白

**原文**：
```
谁家玉笛暗飞声，散入春风满洛城。
此夜曲中闻折柳，何人不起故园情。
```

**翻译**：
是谁家的玉笛在暗暗飘出悠扬的笛声？随着春风传遍整个洛阳城。今夜听到《折杨柳》的曲子，谁能不涌起思念故乡的情感呢？

**创作背景**：
李白客居洛阳时，春夜听到笛声而作。

**主题**：抒发了客居他乡的游子对故乡的思念之情。

**艺术特色**：
- 想象奇特："暗飞声""散入春风"
- 夸张手法："满洛城"
- 用典：折柳——古人送别时折柳相赠，寓"惜别"之意

**重点词语**：
- 暗飞声：笛声悄然飘来
- 洛城：即洛阳（今属河南）
- 折柳：指《折杨柳》曲，乐府横吹曲名
- 故园：故乡

**中考出题概率**：80%

---

### 7.3 《逢入京使》岑参

**原文**：
```
故园东望路漫漫，双袖龙钟泪不干。
马上相逢无纸笔，凭君传语报平安。
```

**翻译**：
向东遥望故园，长路漫漫；两袖被泪水沾湿，怎么也擦不干。在马上与你相逢，没有纸笔写信，只能托你捎个口信，向家人报个平安。

**创作背景**：
天宝八年（749年），岑参赴安西（今新疆库车）途中遇回京使者而作。

**主题**：表现了诗人远涉边塞的思乡怀亲之情。

**艺术特色**：
- 直抒胸臆：直接表达思乡之情
- 细节感人："双袖龙钟泪不干"
- 语言质朴：如话家常

**重点词语**：
- 入京使：回京城长安的使者
- 龙钟：老态龙钟的样子，这里指泪流纵横
- 凭：托，请
- 传语：捎口信

**中考出题概率**：75%

### 7.4 《晚春》韩愈

**原文**：
```
草树知春不久归，百般红紫斗芳菲。
杨花榆荚无才思，惟解漫天作雪飞。
```

**翻译**：
花草树木知道春天即将离去，于是各种红花紫花争奇斗艳、竞相开放。杨花和榆钱没有什么才华情思，只知道像雪花一样漫天飞舞。

**主题**：借晚春景物，表达了对春天的留恋，也蕴含着对人才的思考。

**艺术特色**：
- 拟人手法：赋予草木以人的情感和动作
- 比喻贴切：杨花榆荚比作雪花
- 哲理意味：表面嘲讽杨花榆荚，实则赞美它们不甘寂寞的精神

**重点词语**：
- 不久归：即将离去
- 芳菲：芳香的花草
- 杨花：柳絮
- 榆荚：榆钱
- 才思：才华情思
- 惟解：只知道
- 漫天：满天

**中考出题概率**：75%
