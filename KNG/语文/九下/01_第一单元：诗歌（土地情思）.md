## 知识点1：舒婷《祖国啊，我亲爱的祖国》

### 内容
**作者**：舒婷（当代诗人，朦胧诗派代表）
**出处**：《双桅船》
**写作背景**：1979年，改革开放初期，诗人对祖国深沉的爱与期盼。

**诗歌结构**：
- **第一节**：描绘祖国贫穷落后的历史（"破旧的老水车""熏黑的矿灯"）
- **第二节**：表现人民的苦难与希望（"飞天"袖间未落到地面的花朵）
- **第三节**：展现祖国新生的喜悦（"簇新的理想""古莲的胚芽"）
- **第四节**：抒发献身祖国的誓言（"从我的血肉之躯上去取得"）

**意象分析**：
| 意象 | 象征意义 |
|------|----------|
| 破旧的老水车 | 祖国落后的农业 |
| 熏黑的矿灯 | 祖国艰难的工业 |
| 干瘪的稻穗 | 人民的贫困 |
| 失修的路基 | 祖国的困顿 |
| 淤滩上的驳船 | 祖国的艰难前行 |
| 雪白的起跑线 | 新的开始 |
| 绯红的黎明 | 美好的未来 |

**艺术特色**：
- 运用大量新颖奇特的意象
- 情感层层递进，由悲到喜
- 反复咏叹，增强感染力

**名句**："我是你十亿分之一，是你九百六十万平方的总和"

### 出处
- 教材：九年级下册语文（部编版）
- 页码：第2-4页

### 中考出题概率
90%

### 常见题型
- 意象含义理解题
- 情感变化分析题
- 艺术手法赏析题
- 名句默写与理解题

### 注意事项
- 理解诗歌中意象的象征意义
- 把握情感由低沉到高昂的变化
- 注意朦胧诗的语言特点

## 知识点2：陈毅《梅岭三章》

### 内容
**作者**：陈毅（无产阶级革命家、军事家、诗人）
**出处**：《陈毅诗词选集》
**写作背景**：1936年，陈毅在梅岭被围，以为难以脱险，写下这三首绝命诗。

**三首诗内容**：

**第一章**：
- 内容：回顾革命历程，表达必死决心
- 名句："此去泉台招旧部，旌旗十万斩阎罗"
- 情感：视死如归，革命到底

**第二章**：
- 内容：勉励战友继续斗争
- 名句："南国烽烟正十年，此头须向国门悬"
- 情感：牵挂战友，关怀革命

**第三章**：
- 内容：预言革命胜利，表达乐观信念
- 名句："取义成仁今日事，人间遍种自由花"
- 情感：乐观坚定，理想崇高

**艺术特色**：
- 化用典故（"泉台""阎罗""国门悬""取义成仁"）
- 想象奇特（地下招旧部斩阎罗）
- 气概豪迈，格调高昂

**主题**：表现革命者视死如归的壮烈情怀和革命必胜的坚定信念。

### 出处
- 教材：九年级下册语文（部编版）
- 页码：第5-6页

### 中考出题概率
85%

### 常见题型
- 诗歌内容理解题
- 典故运用分析题
- 情感主旨概括题
- 名句默写题

### 注意事项
- 理解"泉台""阎罗"等词语的象征意义
- 把握三首诗的内在联系
- 背诵三首诗歌全文

## 知识点3：《短诗五首》

### 内容

#### 1. 《月夜》沈尹默
**内容**："我和一株顶高的树并排立着，却没有靠着"
**主题**：表现五四时期青年一代独立不倚的坚强性格和追求思想自由的精神。
**艺术特色**：简洁凝练，富有哲理。

#### 2. 《萧红墓畔口占》戴望舒
**作者**：戴望舒（现代诗人，"雨巷诗人"）
**背景**：1944年，诗人拜谒萧红墓时所作。
**名句**："我等待着，长夜漫漫"
**主题**：表达对亡友的深切怀念和对抗战胜利的期盼。

#### 3. 《断章》卞之琳
**名句**："你站在桥上看风景，看风景人在楼上看你。明月装饰了你的窗子，你装饰了别人的梦。"
**主题**：表达相对观念，世间万物相互依存、相互转化的哲理。
**艺术特色**：构思巧妙，语言精致，富有哲理意味。

#### 4. 《风雨吟》芦荻
**主题**：面对人生风雨，表现出刚毅顽强的精神。
**意象**：风雨、大地、房舍等。

#### 5. 《统一》聂鲁达（智利）
**作者**：巴勃罗·聂鲁达，智利诗人，诺贝尔文学奖获得者。
**主题**：表现对世界统一、和平的向往。

### 出处
- 教材：九年级下册语文（部编版）
- 页码：第7-10页

### 中考出题概率
80%

### 常见题型
- 诗歌主旨理解题
- 名句默写题（特别是《断章》）
- 艺术手法赏析题

### 注意事项
- 背诵《断章》全诗
- 理解《断章》的哲理内涵
- 掌握不同诗歌的风格特点

## 知识点4：《海燕》高尔基

### 内容
**作者**：高尔基（苏联作家，无产阶级文学奠基人）
**出处**：散文诗集《春天的旋律》
**写作背景**：1901年，俄国革命前夕，作者参加彼得堡游行示威后写成。

**象征意义**：
| 形象 | 象征意义 |
|------|----------|
| 海燕 | 英勇无畏的革命先驱者 |
| 暴风雨 | 席卷一切的革命浪潮 |
| 大海 | 广大人民群众的力量 |
| 海鸥、海鸭、企鹅 | 害怕革命的资产阶级自由派 |
| 乌云、闪电 | 沙皇反动势力 |
| 太阳 | 革命的胜利 |

**艺术特色**：
- 象征手法的巧妙运用
- 对比手法的运用（海燕与海鸥等）
- 环境描写的烘托作用
- 散文诗的形式，散文与诗的融合

**名句**："让暴风雨来得更猛烈些吧！"

**主题**：热情歌颂俄国无产阶级革命先驱者坚强无畏的战斗精神，预言革命风暴即将到来。

### 出处

- 页码：第11-14页

90%

### 注意事项

## 知识点5：单元写作——学习扩写

### 内容
**扩写的定义**：在原文的基础上，通过合理的想象和联想，把内容写得更充实、更具体、更生动。

**扩写方法**：
1. **补充细节**：添加人物的外貌、动作、心理、语言描写
2. **补充环境**：增加环境描写，渲染气氛
3. **补充情节**：增加起因、经过、结果的细节
4. **补充议论抒情**：增加作者的感悟和情感

**扩写原则**：
1. **忠于原文**：不改变原文的中心思想和主要内容
2. **合理想象**：想象要符合生活实际和人物性格
3. **突出重点**：详略得当，不能面面俱到
4. **语言生动**：运用修辞手法，增强表现力

**扩写类型**：
- 扩写记叙文
- 扩写议论文（补充论据）
- 扩写说明文（补充说明内容）
- 扩写诗歌（改写成散文）

### 出处
- 教材：九年级下册语文（部编版）
- 页码：第15-16页

### 中考出题概率
75%

### 常见题型
- 扩写写作题
- 片段扩写题

### 注意事项
- 扩写不等于改写，要忠于原文
- 想象要合理，不能脱离实际
- 注意详略，突出重点
