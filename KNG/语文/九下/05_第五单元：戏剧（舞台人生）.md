## 知识点1：《屈原（节选）》郭沫若

### 内容
**作者**：郭沫若（现代作家、诗人、历史学家）
**出处**：历史剧《屈原》第五幕第二场
**体裁**：话剧（历史剧）
**写作背景**：1942年，抗日战争时期，借古讽今。

**剧情背景**：
楚国爱国诗人屈原被南后陷害，被囚禁在东皇太一庙，面对黑暗现实，发出了愤怒的控诉。

**主要人物**：
- **屈原**：爱国诗人，坚持真理，不屈不挠
- **郑詹尹**：奉命毒杀屈原的庙祝，阴险狡猾
- **南后、靳尚**（未出场）：陷害屈原的奸佞

**核心段落——雷电颂**：
屈原呼唤雷电，以风、雷、电象征摧毁黑暗势力的力量。

**象征意义**：
| 形象 | 象征意义 |
|------|----------|
| 风 | 变革社会的力量 |
| 雷 | 反抗压迫的怒吼 |
| 电 | 摧毁黑暗的利剑 |
| 洞庭湖、长江、东海 | 广大人民群众 |
| 东皇太一庙 | 黑暗腐朽的现实 |

**艺术特色**：
1. **独白形式**：以屈原独白为主，情感奔放
2. **象征手法**：风雷电象征革命力量
3. **排比、反复**：增强气势，表达激情
4. **想象奇特**：与宇宙对话，气势磅礴

**主题**：
- 歌颂屈原的爱国精神和高洁品格
- 借屈原影射现实，表达对黑暗势力的反抗
- 抒发对光明和正义的向往

**重点台词**：
- "雷电颂"全段
- "爆炸了吧！爆炸了吧！"
- "我思念那洞庭湖，我思念那长江，我思念那东海"

## 知识点2：《天下第一楼》何冀平

### 内容
**作者**：何冀平（当代剧作家）
**出处**：话剧《天下第一楼》
**体裁**：话剧（京味话剧）

**剧情背景**：
民国初年，北京烤鸭老字号"福聚德"由盛转衰的故事。

**主要人物**：
- **卢孟实**：福聚德掌柜，精明能干，励精图治
- **王子西**：福聚德二掌柜，老成持重
- **常贵**：堂头，善于应酬，忠心耿耿
- **罗大头**：烤鸭师傅，技术高超，脾气倔强
- **克五**：纨绔子弟，好吃懒做
- **唐茂昌、唐茂盛**：东家少爷，不学无术

**主要冲突**：
1. **内部矛盾**：东家少爷与掌柜的矛盾
2. **外部矛盾**：与竞争对手的斗争
3. **人物矛盾**：各人物之间的利益冲突

**剧本结构**：
以福聚德的发展为主线，展现民国初年社会的众生相。

**京味特色**：
- 语言京味儿十足
- 展现老北京风土人情
- 反映老北京商业文化

**主题**：
- 展现老字号"福聚德"的兴衰
- 批判封建家长制和腐朽的少爷习气
- 歌颂劳动人民的智慧和勤劳
- 揭示民族工商业发展的艰难

## 知识点3：《枣儿》孙鸿

### 内容
**作者**：孙鸿（当代剧作家）
**体裁**：话剧（独幕话剧）

**剧情简介**：
在乡间一棵挂满红枣的老树下，一位老人遇到了一个捡枣儿的男孩，一老一少围绕"枣儿"展开了一段温馨的交往。

**主要人物**：
- **老人**：留守老人，儿子进城工作，孤独寂寞
- **男孩**：留守儿童，父亲进城工作，与爷爷相依为命

**"枣儿"的象征意义**：
1. **亲情的象征**：老人对儿子的思念，男孩对父亲的期盼
2. **故乡的象征**：对故土和传统的眷恋
3. **童年的象征**：美好的回忆和纯真的情感

**戏剧冲突**：
- 表面冲突：老人与男孩的对话交流
- 深层冲突：现代化进程中传统与变迁的矛盾
- 情感冲突：留守老人与留守儿童的情感缺失

**艺术特色**：
1. **象征手法**：枣儿、枣树的象征意义
2. **诗化语言**：语言简洁，富有诗意
3. **留白艺术**：结尾开放式，引人深思
4. **以小见大**：通过小事反映社会大问题

**主题**：
- 关注留守老人和留守儿童问题
- 反映城市化进程中农村的变迁
- 呼唤亲情的回归和社会的关怀
- 思考现代化与传统的关系

**社会意义**：
反映了改革开放后，农村劳动力进城务工带来的社会问题——留守老人和留守儿童的情感缺失。

## 知识点4：戏剧文学常识

### 内容
**戏剧的定义**：
戏剧是一种综合性的舞台艺术，由文学、音乐、美术、舞蹈等多种艺术成分组成。

**戏剧的分类**：
1. **按表现形式**：话剧、歌剧、舞剧、戏曲
2. **按题材**：历史剧、现代剧、神话剧
3. **按容量**：多幕剧、独幕剧
4. **按矛盾冲突性质**：悲剧、喜剧、正剧

**戏剧的特点**：
1. **综合性**：多种艺术的综合
2. **舞台性**：受时空限制
3. **冲突性**：矛盾冲突是戏剧的核心
4. **对话性**：以人物对话为主要表现手段

**戏剧的结构**：
- 开端：介绍背景，展开冲突
- 发展：冲突逐渐激化
- 高潮：冲突最激烈的部分
- 结局：冲突的解决

**戏剧语言**：
1. **台词**：人物对话、独白、旁白
2. **舞台说明**：交代场景、动作、表情等
