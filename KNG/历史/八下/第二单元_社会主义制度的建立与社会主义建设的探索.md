# 第二单元 社会主义制度的建立与社会主义建设的探索

> **单元概述**：本单元讲述1953-1976年中国社会主义建设的历程，包括工业化起步、三大改造、社会主义制度建立、艰辛探索等，是八下历史的核心单元，中考分值占比最高。

## 单元知识整合

### 时间轴
```
1953年    1954年     1956年底     1956年    1958年      1966年      1970年       1973年      1976年
  │         │          │           │         │           │           │           │           │
  ▼         ▼          ▼           ▼         ▼           ▼           ▼           ▼           ▼
一五计划  一届人大   三大改造    中共八大   "大跃进"   "文革"    人造卫星    杂交水稻   文革结束
开始      召开       完成        召开      人民公社      开始       发射        培育成功      
```

### 核心线索
- **过渡时期（1949-1956）**："一五"计划+三大改造→社会主义制度建立
- **全面建设时期（1956-1966）**：成就与失误并存（中共八大良好开端→"大跃进"和人民公社化）
- **文革时期（1966-1976）**：十年内乱，教训深刻
