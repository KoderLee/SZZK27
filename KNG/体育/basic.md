# 体育与健康 - 2027年深圳中考

## 考试信息

| 项目 | 内容 |
|:--|:--|
| 分值 | 50分（过程性评价14分 + 现场考试36分） |
| 考试形式 | 过程性评价 + 现场考试 |
| 关键变化/备注 | 增加过程性评价，现场考试项目调整为"三大类各选一项" |

## 评分构成

| 部分 | 分值 | 说明 |
|:--|:--|:--|
| 过程性评价 | 14分 | 平时表现、体质测试等 |
| 现场考试 | 36分 | 三大类各选一项 |

## 现场考试项目（三大类各选一项）

| 类别 | 可选项目 |
|:--|:--|
| 一类 | 中长跑（男生1000米/女生800米）、200米跑 |
| 二类 | 篮球、排球、足球、乒乓球、羽毛球、网球、游泳 |
| 三类 | 立定跳远、一分钟跳绳、实心球、引体向上（男）/仰卧起坐（女） |

## 说明

体育考试贯穿初中三年，七年级开始的过程性评价计入总分，建议尽早重视平时训练和体质测试。
